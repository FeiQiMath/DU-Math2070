    Section 1.6 Separable Equations

    In general, a nonlinear ODE is very difficult to solve. We can only handle a few special types of nonlinear ODEs.
    ðŸ”—

    Subsection 1.6.1 Form of a separable ODE

    In this section, we study the separable ODEs, namely, an ODE of the form:
    \begin{equation*} \dfrac{dy}{dx} = f(y)g(x), \end{equation*}
    i.e., the derivative \(dy/dx\) is a product of two functions, one depends only on \(x\text{,}\) the other depends only on \(y\text{.}\) To solve this ODE, we first separate the variables,
    \begin{equation*} \dfrac{dy}{dx} = f(y)g(x) \Rightarrow \dfrac{dy}{f(y)} = g(x)dx \end{equation*}
    then integrate both sides,
    \begin{equation*} \int \dfrac{dy}{f(y)} = \int g(x)dx \end{equation*}
    The process yields a one-parameter family of implicit solutions.
    ðŸ”—

    Example 1.6.1. Solve the separable ODE \(\dfrac{dy}{dx} = \dfrac{e^x - x}{e^{-y} + y}\).

    • Separate the variables:
      \begin{equation*} (e^x - x)dx = (e^{-y} + y)dy \end{equation*}
      ðŸ”—
    • Integrate both sides:
      \begin{align*} \amp \int (e^x - x)dx = \int (e^{-y} + y)dy\\ \implies \amp e^x - \dfrac{1}{2}x^2 = -e^{-y} + \dfrac{1}{2}y^2 + C \end{align*}
      ðŸ”—
      We thus obtain a one-parameter family of implicit solutions
      ðŸ”—
      ðŸ”—
    ðŸ”—

    Example 1.6.2. Solve the IVP \(\dfrac{dy}{dx} = \dfrac{x^2 + \sin x}{y}, \quad y(0) = 1\).

    • Separate the variables:
      \begin{equation*} ydy = (x^2 + \sin x)dx \end{equation*}
      ðŸ”—
    • Integrate both sides:
      \begin{align*} \amp \int ydy = \int (x^2 + \sin x)dx\\ \implies \amp \dfrac{1}{2}y^2 = \dfrac{1}{3}x^3 - \cos x + C \end{align*}
      ðŸ”—
      We thus obtain a one-parameter family of implicit solutions
      ðŸ”—
      ðŸ”—
    • The initial condition should give us a particular solution
      \begin{equation*} y(0) = 1 \implies \dfrac{1}{2} = 0 - 1 + C \implies C = \dfrac{3}{2} \end{equation*}
      ðŸ”—
      We thus obtain a particular implicit solution
      \begin{equation*} \dfrac{1}{2}y^2 = \dfrac{1}{3}x^3 - \cos x + \dfrac{3}{2} \end{equation*}
      ðŸ”—
      ðŸ”—
    • This solution can actually be made explicit by solving for \(y\) in terms of \(x\text{.}\) Multiply both sides by 2 and take the square root,
      \begin{align*} \amp y^2 = \dfrac{2}{3}x^3 - 2\cos x + 3\\ \amp \implies y = \pm \sqrt{\dfrac{2}{3}x^3 - 2\cos x + 3} \end{align*}
      ðŸ”—
      Since \(y(0) = 1\text{,}\) we should take the positive branch. So
      \begin{equation*} y = \sqrt{\dfrac{2}{3}x^3 - 2\cos x + 3} \end{equation*}
      is the explicit solution of the IVP.
      ðŸ”—
      ðŸ”—
    • The solution makes sense when
      \begin{equation*} \dfrac{2}{3}x^3 - 2\cos x + 3 \geq 0. \end{equation*}
      Solving this inequality is not possible without a computer. So we will leave it as is.
      ðŸ”—
    Conclusion: The solution is
    \begin{equation*} y = \sqrt{\dfrac{2}{3}x^3 - 2\cos x + 3}. \end{equation*}
    It is defined when
    \begin{equation*} \dfrac{2}{3}x^3 - 2\cos x + 3 \geq 0. \end{equation*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Subsection 1.6.2 Implicit solution vs explicit solution

    In this section, we discuss the difference between an implicit solution and an explicit solution.
    ðŸ”—
    • If you are given an ODE without initial values, then a one-parameter family of implicit solutions is sufficient.
      ðŸ”—
    • If you are given an IVP, whenever possible, you should try to get the explicit solution and the interval of definition (the interval where the solution is defined).
      ðŸ”—

    Remark 1.6.3.

    • Suppose we have an initial condition specified for the ODE in ExampleÂ 1.6.1, which determines a definite \(C\text{.}\) But it is impossible to solve for \(y\) in terms of \(x\) to get the explicit solution. In that case, we have to be satisfied with the implicit solution and ask a computer to handle the function values.
      ðŸ”—
    • In ExampleÂ 1.6.2, we managed to solve for \(y\) in terms of \(x\text{,}\) but was not able to solve for the interval of definition by hand. Nevertheless, we can still ask a computer to solve for the interval of definition. For ExampleÂ 1.6.2, the computer would say that the solution makes sense on \([-1.69533, +\infty)\text{.}\) But we should exclude the endpoint \(-1.69533\) and use \((-1.69533, +\infty)\) instead, because the function is not differentiable at the boundary point. Excluding the boundary points and use an open interval is a convention. But exceptions do exist. We shall not require this technical matter in this course.
      ðŸ”—
    • The interval of definition, as its name hints, should consist of only one interval that contains the initial \(x\)-value. In case the solution is defined on multiple intervals, we should pick the interval that contains the initial \(x\)-value.
      ðŸ”—
    ðŸ”—

    Example 1.6.4. Solve the ODE \(y' = xy^3(1 + x^2)^{-1/2}\).

    • Separate the variables:
      \begin{equation*} \dfrac{dy}{dx} = y^3 \cdot \dfrac{x}{\sqrt{1 + x^2}} \implies \dfrac{dy}{y^3} = \dfrac{x}{\sqrt{1 + x^2}}dx \end{equation*}
      ðŸ”—
    • Integrate both sides:
      \begin{equation*} \int \dfrac{dy}{y^3} = \int \dfrac{x}{\sqrt{1 + x^2}}dx \implies -\dfrac{1}{2}y^2 = \sqrt{1 + x^2} + C \end{equation*}
      ðŸ”—
    We thus obtain a one-parameter family of implicit solutions
    ðŸ”—
    ðŸ”—

    Example 1.6.5. Solve the IVP \(y' = xy^3(1 + x^2)^{-1/2}, \, y(\sqrt{3}) = \dfrac{1}{2}\).

    We have obtained a one-parameter family of implicit solutions in ExampleÂ 1.6.4, namely,
    \begin{equation*} -\dfrac{1}{2}y^2 = \sqrt{1 + x^2} + C. \end{equation*}
    We shall continue from here.
    ðŸ”—
    • We determine the constant \(C\) by using the initial condition:
      \begin{equation*} y(\sqrt{3}) = \dfrac{1}{2} \implies -\dfrac{1}{2} \cdot (1/2)^2 = \sqrt{1 + 3} + C \implies C = -4. \end{equation*}
      ðŸ”—
    • The implicit solution we thus obtained is
      \begin{equation*} -\dfrac{1}{2y^2} = \sqrt{1 + x^2} - 4. \end{equation*}
      It is possible to solve an explicit solution. Take the negative reciprocal and simplify,
      \begin{equation*} y^2 = \dfrac{1}{2(4 - \sqrt{1 + x^2})}= \dfrac{1}{8 - 2\sqrt{1 + x^2}}. \end{equation*}
      Now take the square root. Since \(y(\sqrt{3}) = \dfrac{1}{2} \gt 0\text{,}\) we take the positive branch, thus
      \begin{equation*} y = \sqrt{\dfrac{1}{8 - 2\sqrt{1 + x^2}}}. \end{equation*}
      ðŸ”—
    • Solution exists when
      \begin{align*} \amp 8 - 2\sqrt{1 + x^2} \gt 0 \implies 2\sqrt{1+x^2}\lt 8\implies 1+x^2 \lt 16 \\ \implies \amp x^2 \lt 15 \implies x \in (-\sqrt{15}, \sqrt{15}) \end{align*}
      ðŸ”—
    Conclusion: The solution is
    \begin{equation*} y = \sqrt{\dfrac{1}{8 - 2\sqrt{1 + x^2}}}. \end{equation*}
    The interval of definition is \((-\sqrt{15}, \sqrt{15})\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.6.6. Solve the IVP \(y' = xy^3(1 + x^2)^{-1/2}, \, y(\sqrt{3}) = -\dfrac{1}{4}\).

    We have obtained a one-parameter family of implicit solutions in ExampleÂ 1.6.4, namely,
    \begin{equation*} -\dfrac{1}{2}y^2 = \sqrt{1 + x^2} + C. \end{equation*}
    We shall continue from here.
    ðŸ”—
    • We determine the constant \(C\) by using the initial condition:
      \begin{equation*} y(\sqrt{3}) = -\dfrac{1}{4} \implies -\dfrac{1}{2} \cdot (-\dfrac{1}{4})^2 = \sqrt{1 + 3} + C \implies C = -10. \end{equation*}
      ðŸ”—
    • The implicit solution we thus obtained is
      \begin{equation*} -\dfrac{1}{2y^2} = \sqrt{1 + x^2} - 10. \end{equation*}
      It is possible to solve an explicit solution. Take the negative reciprocal and simplify,
      \begin{equation*} y^2 = \dfrac{1}{2(10 - \sqrt{1 + x^2})}= \dfrac{1}{20 - 2\sqrt{1 + x^2}}. \end{equation*}
      Now take the square root. Since \(y(\sqrt{3}) = -\dfrac{1}{4} \lt 0\text{,}\) we take the negative branch, thus
      \begin{equation*} y = -\sqrt{\dfrac{1}{20 - 2\sqrt{1 + x^2}}}. \end{equation*}
      ðŸ”—
    • Solution exists when
      \begin{align*} \amp 20 - 2\sqrt{1 + x^2} \gt 0 \implies 2\sqrt{1+x^2}\lt 20\implies 1+x^2 \lt 100 \\ \implies \amp x^2 \lt 99 \implies x \in (-\sqrt{99}, \sqrt{99}) \end{align*}
      ðŸ”—
    Conclusion: The solution is
    \begin{equation*} y = -\sqrt{\dfrac{1}{20 - 2\sqrt{1 + x^2}}}. \end{equation*}
    The interval of definition is \((-\sqrt{99}, \sqrt{99})\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 1.6.7.

    The explicit solution contains more information than the implicit solution. Indeed, Observe that
    • The positive or negative branch cannot be seen in the implicit solution.
      ðŸ”—
    • The interval of definition cannot be determined by the implicit solution.
      ðŸ”—
    This is why whenever it is possible, we should try to solve for the explicit solution and the interval of definition.
    ðŸ”—
    ðŸ”—

    Remark 1.6.8.

    From ExampleÂ 1.6.5 and ExampleÂ 1.6.6, we also see that the positive or negative branch and the interval of definition depends on the initial condition. Thus these notions make no sense for a one-parameter family of solutions. This is why without the initial condition, we can leave the solution in the implicit form.

    Subsection 1.6.3 Singular solutions

    We saw that \(y' = xy^3(1 + x^2)^{-1/2}\) has a one-parameter family of implicit solutions.
    \begin{equation*} \dfrac{1}{2y^2} = \sqrt{1 + x^2} + C \end{equation*}
    ðŸ”—
    It is not the general solution because the family does not include the solution \(y = 0\text{.}\) Such an extra solution is called a singular solution.
    ðŸ”—
    Another example is
    \begin{equation*} \dfrac{dy}{dx} = x\sqrt{y} \end{equation*}
    which has a one-parameter family of solutions
    \begin{equation*} y = \left(\dfrac{1}{4}x^2 + C\right)^2 \end{equation*}
    while \(y = 0\) is a singular solution.
    ðŸ”—
    Generally, for a separable ODE
    \begin{equation*} \dfrac{dy}{dx} = f(y)g(x), \end{equation*}
    the constant function \(y = a\) is a singular solution for every number \(a\) satisfying \(f(a) = 0.\) (Clearly, \(LHS = a' = 0, \, RHS = f(a)g(x) = 0.\))
    ðŸ”—
    When we separate the variables, we divided both sides by \(f(y).\) Such an operation implicitly required that \(f(y) \neq 0.\) Therefore, in the subsequent one-parameter family of solutions obtained by integration, the constant functions \(y = a\) with \(f(a) = 0\) will not be included in the family.
    ðŸ”—
    Singular solutions are common for nonlinear ODEs.
