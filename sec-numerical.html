    Section 1.9 Numerical Methods

    Numerical Methods
    • Applies to every reasonably formulated initial value problem.
    • Most useful when the analytic methods fail.
    • Main method for IVP in real world engineering practice.

    Subsection 1.9.1

    Euler’s method.
    Idea: Use the direction field to perform linear approximations.
    Recall: \(f(t) \approx f(t_0) + f'(t_0)(t - t_0)\)
    Consider \(y' = f(t, y), \, y(t_0) = y_0,\) we choose a small step-size \(h.\)
    Then we approximate \(y(t_0 + h)\) by \(y(t_0 + h) \approx y(t_0) + y'(t_0)h = y(t_0) + h \cdot f(t_0, y_0).\)
    We read the line element at the point \((t_0, y(t_0 + h)),\) perform the same procedure to get an approximation of \(y(t_0 + 2h).\)
    Repeating the process, we may approximate \(y(t_0 + nh)\) for every \(n = 1, 2, 3, \ldots\)
    Formula: Given the IVP \(y' = f(t, y)\) and step-size \(h,\) set \(t_n = t_0 + nh, \, n = 0, 1, 2, \ldots\)
    If we let \(y_n\) be the approximation of \(y(t_0 + nh),\) then \(y_n\) can be obtained from \(y_{n-1}, t_{n-1}, h\) by
    \(y_n = y_{n-1} + h \cdot f(t_{n-1}, y_{n-1})\)
    Example 1: \(y' = t - y + 1, \, y(1) = 3.\) Use Euler’s method to approximate \(y(3)\) with step-size \(h = 0.5.\)
    Since \(h = 0.5, \, t_0 = 1,\) we compute that
    \(y_n = y_{n-1} + h(t_{n-1} - y_{n-1} + 1)\)
    \(= y_{n-1} + 0.5(t_0 + (n - 1)h - y_{n-1} + 1)\)
    \(= y_{n-1} + 0.5(1 + (n - 1)0.5 - y_{n-1} + 1)\)
    \(= y_{n-1} + 0.5 + 0.25(n - 1)\)
    \(= 0.5y_{n-1} + 0.75 + 0.25n.\)
    We computed that
    \(y_n = 0.5y_{n-1} + 0.75 + 0.25n.\)
    Want: \(y(3),\) i.e., \(t_0 + nh = 3\) \(\Rightarrow 1 + 0.5n = 3 \Rightarrow n = 4.\)
    So \(y(3) \approx 3.125.\)
    General steps:
    1. Formulate the formula and simplify \(y_n\) in terms of \(y_{n-1}, n, h.\)
    2. Determine how many steps you want to compute.
    3. Compute using either a calculator or some software.
    Example 2: \(y' = t + y, \, y(1) = 2.\) Use Euler’s method to approximate \(y(3)\) with step-size \(h = 1.\)
    Since \(h = 1, \, t_0 = 1,\) we compute that
    \(y_n = y_{n-1} + h(t_{n-1} + y_{n-1})\)
    \(= y_{n-1} + 1 \cdot (t_0 + (n - 1)h + y_{n-1})\)
    \(= y_{n-1} + (1 + (n - 1) + y_{n-1})\)
    \(= 2y_{n-1} + n.\)
    We computed that
    \(y_n = 2y_{n-1} + n.\)
    Want: \(y(3),\) i.e., \(t_0 + nh = 3\) \(\Rightarrow 1 + n = 3 \Rightarrow n = 2.\)
    So \(y(3) \approx 12.\)

    Subsection 1.9.2

    Error analysis
    In Example 1, we approximated that \(y(3) \approx 3.125.\)
    The IVP \(y' = t - y + 1, \, y(1) = 3\) can be solved directly.
    The actual solution is \(y(t) = t + 2e^{1-t} \Rightarrow y(3) \approx 3.27067.\)
    The approximation is not very good. However, if we use \(h = \frac{1}{10},\) or \(h = \frac{1}{10,000},\) the approximation will be much better.
    Recall: Taylor’s theorem:
    \(y(t_0 + h) = y(t_0) + hy'(t_0) + \frac{h^2}{2}y''(t_0) + \ldots\)
    In the Euler’s method, we only kept the first two terms.
    The dropped terms can be expressed using Lagrange remainder:
    \(Error = \frac{1}{2}h^2y''(t_1), \, t_1 \in (t_0, t_0 + h)\)
    1. If \(y''(t) \gt 0\) on \((t_0, t_0 + h),\) Euler’s method gives an underestimate of \(y(t_0 + h),\) i.e., the estimate \(y_1\) is smaller than the actual solution \(y(t_0 + h).\)
    2. If \(y''(t) \lt 0\) on \((t_0, t_0 + h),\) Euler’s method gives an overestimate of \(y(t_0 + h),\) i.e., the estimate \(y_1\) is larger than the actual solution \(y(t_0 + h).\)
    3. If \(y''(t)\) changes sign. In that case, we can’t tell.
    4. The formula describes the error in one step, thus is called the local truncation error. It is proportional to \(h^2.\) In particular, lowering \(h\) means lowering the error.
    5. The final error, aka global truncation error, will accumulate.
    6. In general, if the local truncation \(\sim h^k\text{,}\) then global truncation error \(\sim h^{k-1}.\) For Euler’s method, the GTE \(\sim h.\)
    Example 1: \(y' = t - y + 1, \, y(1) = 3.\)
    We estimated that \(y(3) \approx 3.125.\) Do we have an overestimate or an underestimate?
    To find \(y''\text{,}\) we take the derivative on both sides of the ODE:
    \(y'' = t' - y' + 1' = 1 - (t - y + 1) = -t + y\)
    Within \([1, 3],\) we can estimate \(y''\) using the values of each step. Recall that \(y(1) = 3, \, y(1.5) \approx 2.5, \, y(2) \approx 2.5, \, y(2.5) \approx 2.75, \, y(3) \approx 3.125.\)
    \(\Rightarrow y''(1.5) \approx -1.5 + 2.5 = 1, \, y''(2) \approx -2 + 2.5 = 0.5, \, y''(2.5) \approx -2.5 + 2.75 = 0.25, \, y''(3) \approx -3 + 3.125 = 0.125.\)
    All positive. It is likely that we have an underestimate.
    Warning: This process is not fully rigorous. It provides some evidence that is usually good enough in engineering practices.
    Example 1: \(y' = t - y + 1, \, y(1) = 3.\)
    We estimated that \(y(3) \approx 3.125.\) If the error of \(y(3)\) is 0.2 with step-size \(h = 0.5,\) what would the error be if step-size \(h = 0.01?\)
    We know that the global truncation error is proportional to \(h,\) i.e., \(E = Kh.\)
    We have \(E = 0.2\) when \(h = 0.5.\) So \(0.2 = K \cdot 0.5.\)
    When \(h = 0.01,\) we have \(E = K \cdot 0.01 = \frac{0.2}{0.5} \cdot 0.01 = 0.004.\)
    Remark: We may use this method to decide the step-size so that the error is controlled within a certain precision level.

    Subsection 1.9.3

    • Improved Euler’s method:\(y' = f(t, y), \, y(t_0) = y_0\)
      • \(\displaystyle t_n = t_0 + nh\)
      • \(\displaystyle y^*_n = y_{n-1} + h \cdot f(t_{n-1}, y_{n-1})\)
      • \(\displaystyle y_n = y_{n-1} + h \cdot \frac{1}{2}(f(t_{n-1}, y_{n-1}) + f(t_n, y^*_n))\)
      • Local truncation error \(\sim h^3\text{,}\) Global truncation error \(\sim h^2\)
    • Runge-Kutta method (RK4)
      • \(\displaystyle t_n = t_0 + nh\)
      • \(\displaystyle K_1 = f(t_{n-1}, y_{n-1}), \, K_2 = f(t_{n-1} + \frac{1}{2}h, y_{n-1} + \frac{1}{2}K_1h)\)
      • \(\displaystyle K_3 = f(t_{n-1} + \frac{1}{2}h, y_{n-1} + \frac{1}{2}K_2h)\)
      • \(\displaystyle K_4 = f(t_{n-1} + h, y_{n-1} + K_3)\)
      • \(\displaystyle y_n = y_{n-1} + \frac{h}{6}(K_1 + 2K_2 + 2K_3 + K_4)\)
      • Local truncation error \(\sim h^5\text{,}\) Global truncation error \(\sim h^4\)
