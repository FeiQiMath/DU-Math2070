    Section 2.1 General Theories of Second-order Linear ODEs

    From Section 1.4, the standard form of a second-order linear ODE is
    \begin{equation*} y'' + p(t)y' + q(t)y = g(t). \end{equation*}
    If \(g(t) = 0\text{,}\) then the ODE
    \begin{equation*} y'' + p(t)y' + q(t)y = 0 \end{equation*}
    is said to be homogeneous. Otherwise, we say the ODE is nonhomogeneous.
    Unlike the first-order ODE where a formula gives the general solution (see Section 1.5), for second- and higher-order linear ODEs, we do not know how to solve them in general. In this course we will only focus on some special types of second-order linear ODEs that appear frequently in science and engineering courses.
    We start with some general theories.

    Subsection 2.1.1 Existence and Uniqueness Theorem

    Theorem 2.1.1. Existence and Uniqueness Theorem.

    For the IVP
    \begin{equation*} y'' + p(t)y' + q(t)y = g(t), \, y(t_0) = y_0, \, y'(t_0) = y_1 \end{equation*}
    If \(p(t), q(t), g(t)\) are continuous in an interval \((a, b)\) that contains \(t_0\text{,}\) then the IVP has a unique solution on \((a, b)\text{.}\)
    The proof of this theorem is beyond the level of this course. We shall focus on using it. Just like in Subsection 1.7.2, we may use the theorem to find the interval of existence.

    Example 2.1.2. Find the interval of existence of the IVP \(t^2y'' - 4ty' + 4y = 0, \, y(1) = 1, \, y'(1) = 4\).

    • Standard form: \(y'' - \frac{4}{t}y' + \frac{4}{t^2}y = 0\)
    • Singularity: \(t = 0.\)
    Interval of existence: \((0, \infty).\)

    Remark 2.1.3.

    In what follows, all the results are supposed to hold on the interval of existence.

    Subsection 2.1.2 Principle of Superposition

    Theorem 2.1.4. Principle of Superposition.

    1. If \(y_1, y_2\) are solutions of a 2nd-order linear homogeneous ODE
      \begin{equation*} y'' + p(t)y' + q(t)y = 0 \end{equation*}
      then for every real number \(c_1, c_2\text{,}\) the function
      \begin{equation*} c_1y_1 + c_2y_2 \end{equation*}
      is also a solution.
    2. In addition, if \(y_1, y_2\) are linearly independent, i.e., if \(y_1\) and \(y_2\) are not proportional to each other, then the general solution of the ODE is
      \begin{equation*} y = c_1y_1 + c_2y_2 \end{equation*}

    Example 2.1.5. Check that \(y_1 = e^{3t}, \, y_2 = e^{-t}\) are solutions of the ODE \(y'' - 2y' - 3y = 0.\) What does the principle of superposition tell us?

    1. \(y_1 = e^{3t}, \, y_2 = e^{-t}\) are solutions:
      \begin{equation*} y_1'' - 2y_1' - 3y_1 = 9e^{3t} - 2 \cdot 3e^{3t} - 3e^{3t} = 0. \end{equation*}
      \begin{equation*} y''_2 - 2y'_2 - 3y_2 = e^{-t} = e^{-t} - 2 \cdot (-e^{-t}) - 3e^{-t} = 0. \end{equation*}
    2. \(Y(t) = 20e^{3t} - 99e^{-t} = 20y_1 - 99y_2\) is also a solution.
      \begin{align*} \amp Y'' - 2Y' - 3Y\\ = \amp (20y_1 - 99y_2)'' - 2(20y_1 - 99y_2)' - 3(20y_1 - 99y_2)\\ = \amp 20y''_1 - 99y''_2 - 2(20y_1' - 99y_2') - 3(20y_1 - 99y_2)\\ = \amp 20y''_1 - 99y''_2 - 2 \cdot 20y_1' + 2 \cdot 99y_2' - 3 \cdot 20y_1 + 3 \cdot 99y_2\\ = \amp 20(y''_1 - 2y'_1 - 3y_1) - 99(y''_2 - 2y_2' - 3y_2)\\ = \amp 20 \cdot 0 - 99 \cdot 0 = 0. \end{align*}
    3. \(Y(t) = c_1e^{3t} + c_2e^{-t} = c_1y_1 + c_2y_2\) is also a solution.
      \begin{align*} \amp Y'' - 2Y' - 3Y\\ = \amp (c_1y_1 + c_2y_2)'' - 2(c_1y_1 + c_2y_2)' - 3(c_1y_1 + c_2y_2)\\ = \amp c_1y''_1 + c_2y''_2 - 2 \cdot (c_1y'_1 + c_2y'_2) - 3 \cdot (c_1y_1 + c_2y_2)\\ = \amp c_1y''_1 + c_2y''_2 - 2 \cdot c_1y'_1 - 2 \cdot c_2y'_2 - 3 \cdot c_1y_1 - 3 \cdot c_2y_2\\ = \amp c_1(y''_1 - 2y'_1 - 3y_1) + c_2(y''_2 - 2y'_2 - 3y_2)\\ = \amp c_1 \cdot 0 + c_2 \cdot 0 = 0. \end{align*}
    4. We check that
      \begin{equation*} \dfrac{y_1}{y_2} = \dfrac{e^{3t}}{e^{-t}} = e^{4t} \end{equation*}
      that is not a constant. So \(y_1 = e^{3t}\) and \(y_2 = e^{-t}\) are linearly independent, the theorem implies \(y(t) = c_1e^{3t} + c_2e^{-t}\) is the general solution.

    Remark 2.1.6.

    The proof of the first part of the superposition principle is not much more difficult than what we did in the example above. I’ll leave it as an exercise. The second part, however, is a consequence of Theorem 2.1.1. The proof is not difficult, but for the sake of brevity, we shall not prove it here.

    Subsection 2.1.3 Wronskian for linear independence

    Theorem 2.1.7. Wronskian for linear independence.

    If \(y_1, y_2\) are solutions of the 2nd-order linear homogeneous ODE
    \begin{equation*} y'' + p(t)y' + q(t)y = 0, \end{equation*}
    on an interval where the existence and uniqueness theorem holds, then \(y_1, y_2\) are linearly independent if and only if the Wronskian
    \begin{equation*} (W(y_1, y_2))(t) = \begin{vmatrix} y_1(t) \amp y_2(t) \\ y'_1(t) \amp y'_2(t) \end{vmatrix} \end{equation*}
    is nonzero on the interval.

    Fact 2.1.8.

    It suffices that \((W(y_1, y_2))(t) \neq 0\) on one point of the interval. The proof is not difficult. But for the sake of brevity, we shall not prove it here.

    Example 2.1.9. \(t^2y'' - 4ty' + 6y = 0, \, y_1 = t^2, \, y_2 = t^3.\).

    \(W(y_1, y_2) = \begin{vmatrix} t^2 \amp t^3 \\ 2t \amp 3t^2 \end{vmatrix} = t^4 \neq 0 \text{ as long as } t \neq 0.\)
    So \(y_1 = t^2, \, y_2 = t^3\) are linearly independent.

    Example 2.1.10. \(y'' - 9y = 0, \, y_1 = e^{3t}, \, y_2 = 5e^{3t}.\).

    \(W(y_1, y_2) = \begin{vmatrix} e^{3t} \amp 5e^{3t} \\ 3e^{3t} \amp 15e^{3t} \end{vmatrix} = 15e^{6t} - 15e^{6t} = 0.\)
    So \(y_1 = e^{3t}, \, y_2 = 5e^{3t}\) are linearly dependent.

    Example 2.1.11. \(ay'' + by' + cy = 0, \, y_1 = e^{r_1 t}, \, y_2 = e^{r_2 t}, \, r_1 \neq r_2.\).

    \(W(y_1, y_2) = \begin{vmatrix} e^{r_1 t} \amp e^{r_2 t} \\ r_1 e^{r_1 t} \amp r_2 e^{r_2 t} \end{vmatrix} = (r_2 - r_1)e^{(r_1 + r_2)t}.\)
    \(r_1 \neq r_2 \Rightarrow W(y_1, y_2) \neq 0 \Rightarrow \text{Linear independence.}\)

    Example 2.1.12. \(y'' - \cot(2t)y' = 0, \, y_1 = \sin^2 t, \, y_2 = \cos(2t) - 1.\).

    \begin{align*} W(y_1, y_2) = \amp \begin{vmatrix} \sin^2 t \amp \cos(2t) - 1 \\ 2 \sin t \cos t \amp -2 \sin(2t) \end{vmatrix}\\ = \amp -2 \sin^2 t \sin(2t) - (\cos(2t) - 1)2 \sin t \cos t. \end{align*}
    Set \(t = \frac{\pi}{4}\text{.}\) Then
    \begin{equation*} W(y_1, y_2) = -2 \cdot \frac{1}{2} \cdot 1 - (0 - 1) \cdot 2 \cdot \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{2}}{2} = 0. \end{equation*}
    So \(y_1, y_2\) are linearly dependent.
