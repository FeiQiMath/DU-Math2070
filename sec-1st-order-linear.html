    Section 1.5 First-order Linear ODEs

    Theorem 1.5.1.

    Consider a first-order linear ODE in its standard form
    \begin{equation*} y' + p(t)y = g(t). \end{equation*}
    We solve it by first calculating the integrating factor
    \begin{equation*} \mu(t) = e^{\int p(t)dt}, \end{equation*}
    then the formula of the general solution is
    \begin{equation*} y(t) = \frac{\int \mu(t)g(t)dt}{\mu(t)}. \end{equation*}
    How comes the formula? Observe that
    \begin{equation*} e^{\int p(t)dt}(y' + p(t)y) = \left(e^{\int p(t)dt}y\right)'. \end{equation*}
    In general, any function \(\mu(t)\) making
    \begin{equation*} \mu(t)(y'(t) + p(t)y(t)) = (\mu(t)y(t))' \end{equation*}
    is called an integrating factor. To satisfy this equation, we calculate that
    \begin{equation*} RHS = \mu'(t)y(t) + \mu(t)y'(t) \end{equation*}
    So \(LHS = RHS\) means
    \begin{align*} \amp \mu(t)y'(t) + \mu(t)p(t)y(t) = \mu'(t)y(t) + \mu(t)y'(t)\\ \implies \amp \mu(t)p(t)y(t) = \mu'(t)y(t) \Rightarrow \mu'(t) = \mu(t)p(t)\\ \implies \amp \frac{\mu'(t)}{\mu(t)} = p(t) \Rightarrow \ln|\mu(t)| = \int p(t)dt\\ \implies \amp \mu(t) = e^{\int p(t)dt}. \end{align*}
    Now that we found the \(\mu(t)\text{,}\) Recall that the ODE we intend to solve is
    \begin{align*} \amp y'(t) + p(t)y(t) = g(t)\\ \implies \amp \mu(t)(y'(t) + p(t)y(t)) = \mu(t)g(t). \end{align*}
    Thus, we conclude that
    \begin{equation*} (\mu(t)y(t))' = \mu(t)g(t). \end{equation*}
    Integrate both sides:
    \begin{align*} \amp \mu(t)y(t) = \int \mu(t)g(t)dt\\ \implies \amp y(t) = \frac{\int \mu(t)g(t)dt}{\mu(t)}. \end{align*}

    Remark 1.5.2.

    • This formula ONLY works for the standard form. Before using it, get the standard form first.
    • When integrating \(p(t)\text{,}\) no need to worry about the constant or the absolute value. They make no difference.
    • The arbitrary constant in the general solution appears as a part of the numerator, not outside of the fraction. It might be helpful to write the formula of the general solution as
      \begin{equation*} y(t) = \frac{\int \mu(t)g(t)dt + C}{\mu(t)}. \end{equation*}
      From this formula, it is clear that \(C\) is a pure constant only when \(\mu(t)\) is a constant. This case is very rare, and only happens when \(p(t)=0\text{.}\)

    Example 1.5.3. Solve \(y' + 2y = e^{3t}, \quad y(0) = 3.\).

    • Already in standard form.
    • Integrating factor: \(\mu(t) = e^{\int 2dt} = e^{2t}.\)
    • General solution:
      \begin{align*} \amp y(t) = \frac{\int e^{2t}e^{3t}dt}{e^{2t}} = \frac{\int e^{5t}dt}{e^{2t}} = e^{-2t}\left(\frac{1}{5}e^{5t} + C\right).\\ \implies \amp y(t) = \frac{1}{5}e^{3t} + Ce^{-2t}. \end{align*}
    Digression: What if we keep the constant in the integrating factor?
    • Integrating factor: \(\mu(t) = e^{\int 2dt + D} = e^{2t + D} = e^{2t} e^{D}.\)
    • General solution:
      \begin{align*} y(t) \amp = \frac{\int e^{2t} e^{3t} e^{D} dt}{e^{2t} e^{D}} = \frac{\int e^{5t} \cancel{e^D} dt}{e^{2t} \cancel{e^D}} = \frac{\frac{1}{5} e^{5t} + C}{e^{2t}}\\ \amp = \frac{1}{5} e^{3t} + C e^{-2t}. \end{align*}
      So the extra constant in the integrating factor does not make any difference!
    Coming back to the problem, we have obtained the general solution. But we are not done yet. We are given an IVP, so we need to take care of the initial condition.
    • Determine \(C\) by the initial condition:
      \begin{equation*} y(0) = 3 \Rightarrow \frac{1}{5} + C = 3 \Rightarrow C = \frac{14}{5}. \end{equation*}
    Conclusion: The solution of the IVP is
    \begin{equation*} y(t) = \frac{1}{5} e^{3t} + \frac{14}{5} e^{-2t} \end{equation*}

    Example 1.5.4. Solve \(ty' - y = t^3, \quad y(1) = 0.\).

    • Standard form: \(y' - \frac{1}{t}y = t^2.\)
    • Integrating factor: \(\mu(t) = e^{-\int \frac{1}{t} dt} = e^{-\ln |t|} = \frac{1}{t}.\)
      Warning! Saying \(e^{-\ln t} = -t\) commits the crime of abusing algebra, which is punishable by banishment to the STEM.
    • General solution:
      \begin{align*} y(t) \amp = \frac{\int \frac{1}{t} \cdot t^2 dt + C}{\frac{1}{t}} = t \left( \int dt + C \right)\\ \amp = t \left( \frac{1}{2}t^2 + C \right) = \frac{1}{2}t^3 + Ct. \end{align*}
    • Determine \(C\) by initial value:
      \begin{equation*} y(1) = 0 \Rightarrow \frac{1}{2} + C = 0 \Rightarrow C = -\frac{1}{2}. \end{equation*}
    • Solution of the IVP:
      \begin{equation*} y(t) = \frac{1}{2}t^3 - \frac{1}{2}t. \end{equation*}

    Example 1.5.5. Solve \((\sin t) y' + (\cos t) y = \sin^2 t, \quad 0 \lt t \lt \pi.\).

    • Standard form: \(y' + \frac{\cos t}{\sin t} y = \sin t.\)
    • Integrating factor: \(\mu(t) = e^{\int \frac{\cos t}{\sin t} dt}\)
      \begin{gather*} \int \frac{\cos t}{\sin t} dt = \int \frac{d(\sin t)}{\sin t} = \ln |\sin t|.\\ \implies \mu(t) = e^{\ln |\sin t|} = \sin t. \end{gather*}
      Note that \(0 \lt t \lt \pi\Rightarrow \sin t \gt 0, | \sin t | = \sin t.\)
    • General solution: \(y(t) = \dfrac{\int \sin^2 t \, dt + C}{\sin t}\)
      We calculate the integral in the numerator:
      \begin{align*} \int \sin^2 t \, dt \amp = \int \frac{1 - \cos 2t}{2} \, dt = \frac{1}{2} \int (1 - \cos 2t) \, dt\\ \amp = \frac{1}{2} t - \frac{1}{4} \sin 2t + C. \end{align*}
      Thus the general solution is of the form
      \begin{align*} y(t) \amp = \frac{\frac{1}{2}t - \frac{1}{4}\sin 2t + C}{\sin t} = \frac{t}{2\sin t} - \frac{\sin 2t}{4\sin t} + \frac{C}{\sin t}\\ \amp = \frac{t}{2\sin t} - \frac{\cos t}{2} + \frac{C}{\sin t}, \end{align*}
      where in the last step we used the identity \(\sin 2t = 2 \sin t \cos t\text{.}\)

    Example 1.5.6. Solve \(ty' + 2y = t(\ln 3t)^2.\).

    • Standard form: \(y' + \frac{2}{t}y = (\ln 3t)^2.\)
    • Integrating factor: \(\mu(t) = e^{\int \frac{2}{t}dt} = e^{2 \ln t} = t^2.\)
    • General solution:
      \begin{equation*} y(t) = \frac{\int t^2(\ln 3t)^2dt + C}{t^2}. \end{equation*}
      We calculate the integral in the numerator:
      \begin{align*} \int t^2(\ln 3t)^2 dt \amp = \frac{1}{3}t^3(\ln 3t)^2 - \int \frac{1}{3}t^3 \cdot 2\ln 3t \cdot \frac{1}{3} \cdot 3 dt\\ \amp = \frac{1}{3}t^3(\ln 3t)^2 - \frac{2}{3}\int t^2 \ln 3t \, dt\\ \amp= \frac{1}{3}t^3(\ln 3t)^2 - \frac{2}{3}\left(\frac{1}{3}t^3 \ln 3t - \int \frac{1}{3}t^3 \cdot \frac{1}{3t} \cdot 3 dt\right)\\ \amp= \frac{1}{3}t^3(\ln 3t)^2 - \frac{2}{9}t^3 \ln 3t + \frac{2}{9} \cdot \frac{1}{3}t^3 + C\\ \amp= \frac{1}{3}t^3(\ln 3t)^2 - \frac{2}{9}t^3 \ln 3t + \frac{2}{27}t^3 + C \end{align*}
      So we conclude the general solution as
      \begin{align*} y(t) \amp = \frac{\frac{1}{3}t^3(\ln 3t)^2 - \frac{2}{9}t^3 \ln 3t + \frac{2}{27}t^3 + C}{t^2}\\ \amp = \frac{1}{3}t(\ln 3t)^2 - \frac{2}{9}t \ln 3t + \frac{2}{27}t + \frac{C}{t^2} \end{align*}

    Example 1.5.7. Solve \(y' + y = \cos 2t.\).

    • Already in standard form.
    • Integrating factor: \(\mu(t) = e^{\int dt} = e^t.\)
    • General solution:
      \begin{align*} y(t) \amp = \dfrac{\int e^t \cos 2t \, dt }{e^{t}} \\ \amp =\cdots = \frac{1}{5}e^t \cos 2t + \frac{2}{5}e^t \sin 2t + C e^{-t} \end{align*}
      How to calculate the integral?
      \begin{align*} \int e^t \cos 2t \, dt \amp = e^t \cos 2t - \int e^t (-2 \sin 2t) \, dt\\ \amp = e^t \cos 2t + 2 \int e^t \sin 2t \, dt\\ \amp = e^t \cos 2t + 2\left(e^t \sin 2t - \int e^t (2 \cos 2t) \, dt\right)\\ \amp = e^t \cos 2t + 2e^t \sin 2t - 4 \int e^t \cos 2t \, dt \end{align*}
      So we have
      \begin{align*} \amp 5 \int e^t \cos 2t \, dt = e^t \cos 2t + 2e^t \sin 2t+C\\ \implies \amp \int e^t \cos 2t \, dt = \frac{1}{5} e^t \cos 2t + \frac{2}{5} e^t \sin 2t+C \end{align*}
      Then we conclude the general solution as
      \begin{align*} y(t) \amp =\dfrac{\frac{1}{5} e^t \cos 2t + \frac{2}{5} e^t \sin 2t+C }{e^{t}}\\ \amp = \frac{1}{5} e^t \cos 2t + \frac{2}{5} e^t \sin 2t + C e^{-t} \end{align*}

    Example 1.5.8. Solve \(y' - (\tan t)y = \sec^2 t.\).

    • Already in standard form.
    • Integrating factor: \(\mu(t) = e^{-\int \tan t \, dt} = e^{\ln |\cos t|} = \cos t.\)
      Here we ignored the absolute value (why can we do that?).
    • General solution:
      \begin{align*} y(t) \amp = \frac{\int \cos t \cdot \sec^2 t \, dt + C}{\cos t} = \frac{\int \sec t \, dt + C}{\cos t}\\ \amp = \frac{\ln |\sec t + \tan t| + C}{\cos t} \end{align*}
      Please make sure you know how to integrate \(\sec t\text{.}\)

    Example 1.5.9. Solve
    \begin{equation*} y' + y = g(t), \, y(0) = 0, \, g(t) = \begin{cases} 1 \amp 0 \leq t \leq 1 \\ 0 \amp t \gt 1 \end{cases}\text{.} \end{equation*}
    Find the solution of the IVP that is continuous everywhere.

    General principle: Solve the ODE on each individual interval. Determine the constants by either the initial condition or continuity at the boundary .
    On the interval \([0,1], \, y' + y = 1.\)
    • Integrating factor: \(\mu(t) = e^{\int dt} = e^t.\)
    • General solution:
      \begin{equation*} y(t) = \frac{\int e^t \cdot 1 \, dt + C}{e^t} = \frac{e^t + C}{e^t} = 1 + Ce^{-t}. \end{equation*}
    • Initial condition \(y(0) = 0 \Rightarrow 1 + Ce^0 = 0 \Rightarrow C = -1.\)
    • We conclude that on \([0,1], \, y(t) = 1 - e^{-t}.\)
    On the interval \((1, \infty), \, y' + y = 0.\)
    • Integrating factor: \(\mu(t) = e^{\int dt} = e^t.\)
    • General solution:
      \begin{equation*} y(t) = \frac{\int e^t \cdot 0 \, dt + C}{e^t} = \frac{C}{e^t} = Ce^{-t}. \end{equation*}
    • On \((1, \infty), \, y(t) = Ce^{-t}.\)
    So the solution of the IVP is
    \begin{equation*} y(t) = \begin{cases} 1 - e^{-t} \amp t \in [0,1] \\ Ce^{-t} \amp t \in (1,\infty) \end{cases} \end{equation*}
    To make sure \(y(t)\) is continuous, we need \(\lim_{t \to 1^-} y(t) = \lim_{t \to 1^+} y(t).\text{.}\) This is to say \(1 - e^{-1} = Ce^{-1} \Rightarrow C = (e - 1).\)
    Conclusion: the continuous solution of the IVP is
    \begin{equation*} y(t) = \begin{cases} 1 - e^{-t} \amp t \in [0,1] \\ (e - 1)e^{-t} \amp t \in (1,\infty) \end{cases} \end{equation*}
