    Section 2.4 Second-order Linear ODEs with Constant Coefficients - Complex Roots

    Subsection 2.4.1 The case of (distinct) complex roots

    Consider the ODE
    \begin{equation*} ay'' + by' + cy = 0 \end{equation*}
    where \(a, b, c\) are constant numbers. We tried the function \(y = e^{rt}\) and got the auxiliary equation
    \begin{equation*} ar^2 + br + c = 0. \end{equation*}
    If the quadratic equation has two complex roots, then these two roots are distinct and can be written as
    \begin{equation*} r_1 = \alpha + i\beta, \, r_2 = \alpha - i\beta. \end{equation*}
    In this case, we obtain two complex solutions
    \begin{equation*} \tilde{y}_1 = e^{(\alpha + i\beta)t}, \, \tilde{y}_2 = e^{(\alpha - i\beta)t}. \end{equation*}
    Using Eulerâ€™s formula, we may write them as
    \begin{equation*} \tilde{y}_1 = e^{\alpha t} \cos \beta t + i e^{\alpha t} \sin \beta t, \, \tilde{y}_2 = e^{\alpha t} \cos \beta t - i e^{\alpha t} \sin \beta t. \end{equation*}
    Note that \(\cos(-\theta) = \cos \theta, \, \sin(-\theta) = -\sin \theta.\) Note also that \(y_1, y_2\) are conjugate to each other.
    ðŸ”—

    Theorem 2.4.1.

    If \(y(t) = u(t) + iv(t)\) is a complex solution of
    \begin{equation*} y'' + p(t)y' + q(t)y = 0, \end{equation*}
    then both the real part \(u(t)\) and the imaginary part \(v(t)\) are solutions of the ODE as well.
    ðŸ”—
    ðŸ”—
    Proof. Note that
    \begin{equation*} y = u + iv \implies y' = u' + iv', \, y'' = u'' + iv'' \end{equation*}
    Therefore,
    \begin{align*} \amp y'' + py' + qy = 0\\ \implies \amp (u'' + iv'') + p(u' + iv') + q(u + iv) = 0\\ \implies \amp (u'' + pu' + qu) + i(v'' + pv' + qv) = 0\\ \implies \amp u'' + pu' + qu = 0, \, v'' + pv' + qv = 0 \end{align*}
    Recall that
    \begin{equation*} a + bi = c + di \Leftrightarrow a = c, \, b = d. \end{equation*}
    So we conclude that
    \begin{equation*} u'' + pu' + qu = 0, \, v'' + pv' + qv = 0. \end{equation*}
    Therefore, both \(u\) and \(v\) are solutions.
    ðŸ”—

    Theorem 2.4.2.

    If the auxiliary equation of \(ay'' + by' + cy = 0\) has complex solutions \(\alpha + i\beta, \, \alpha - i\beta,\) then the general solution of the ODE is
    \begin{equation*} y = C_1 e^{\alpha t} \cos(\beta t) + C_2 e^{\alpha t} \sin(\beta t). \end{equation*}
    ðŸ”—
    Proof. We already have the complex solutions
    \begin{equation*} \tilde{y}_1 = e^{\alpha t} \cos \beta t + i e^{\alpha t} \sin \beta t, \, \tilde{y}_2 = e^{\alpha t} \cos \beta t - i e^{\alpha t} \sin \beta t. \end{equation*}
    TheoremÂ 2.4.1 implies that
    \begin{equation*} y_1 = e^{\alpha t} \cos \beta t, \, y_2 = e^{\alpha t} \sin \beta t \end{equation*}
    are both solutions. Note that
    \begin{align*} W(y_1, y_2) = \amp \begin{vmatrix} e^{\alpha t} \cos \beta t \amp e^{\alpha t} \sin \beta t \\ \end{vmatrix}\\ = \amp \begin{vmatrix} e^{\alpha t} \cos \beta t \amp e^{\alpha t} \sin \beta t \\ \alpha e^{\alpha t} \cos \beta t - e^{\alpha t} \sin \beta t \amp \alpha e^{\alpha t} \sin \beta t + e^{\alpha t} \cos \beta t \end{vmatrix}\\ = \amp e^{2\alpha t} (\cos^2 \beta t + \sin^2 \beta t) = e^{2\alpha t} \neq 0. \end{align*}
    So \(y_1, y_2\) are linearly independent. The conclusion follows.

    Subsection 2.4.2 Oscillation Patterns

    Fact 2.4.6.

    For real numbers \(A, B\) and \(\theta,\)
    \begin{equation*} A \cos \theta + B \sin \theta = R \cos(\theta - \phi), \end{equation*}
    with
    \begin{equation*} R = \sqrt{A^2 + B^2}, \, \phi =\text{ angle determined by }(A, B). \end{equation*}
    In other words, any linear combination of sine and cosine functions of the same angle can be rewritten as a single cosine function, representing an oscillation with amplitude \(R\) and phase \(\phi.\)
