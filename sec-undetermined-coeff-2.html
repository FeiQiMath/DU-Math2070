    Section 2.8 Method of Undetermined Coefficients - Final Ansatz

    The case when the first ansatz fails
    Example 1. \(y'' - y' - 2y = e^{2t}\)
    Complementary solution: \(y = C_1e^{-t} + C_2e^{2t}.\)
    If we try \(Y = Ae^{2t},\) then \(Y' = 2Ae^{2t}, Y'' = 4Ae^{2t}.\)
    \(Y'' - Y' - 2Y = 2Ae^{2t} - 2Ae^{2t} - 2Ae^{2t} = 0.\)
    There is no way for \(Y'' - Y' - 2Y = e^{2t}.\)
    Why does this happen? Notice that \(e^{2t}\) appears as a part of the complementary solution while my ansatz \(Ae^{2t}\) is its multiple. When this happens, the first try ansatz fails.
    To find a particular solution, we multiply our ansatz by \(t\) and try again. \(Y = Ate^{2t}, Y' = Ae^{2t} + 2Ate^{2t}, Y'' = 4Ae^{2t} + 4Ate^{2t}.\)
    \(Y'' - Y' - 2Y = 4Ae^{2t} + 4Ate^{2t} - (Ae^{2t} + 2Ate^{2t}) - 2Ate^{2t}\)
    \(= 3Ae^{2t} = e^{2t} \implies A = \dfrac{1}{3} \implies Y = \dfrac{1}{3}te^{2t}.\)
    Example 2. \(y'' - 2y' + y = 2te^t.\)
    Complementary solution: \(y = C_1e^t + C_2te^t.\)
    The first try ansatz is \(Y = e^t(At + B) = Ate^t + Be^t.\)
    This ansatz clearly fails, as \(Y\) has the same structure as the complementary solution. We need to multiply by \(t.\)
    \(Y = te^t(At + B) = (At^2 + Bt)e^t.\)
    Then \(Y' = (2At + B)e^t + (At^2 + Bt)e^t,\)
    \(Y'' = 2Ae^t + 2(2At + B)e^t + (At^2 + Bt)e^t.\)
    \(Y'' - 2Y' + Y\)
    \(= 2Ae^t + 2(2At + B)e^t + (At^2 + Bt)e^t\)
    \(- 2((2At + B)e^t + (At^2 + Bt)e^t) + (At^2 + Bt)e^t\)
    \(= 2Ae^t.\)
    We do not have \(te^t\) terms on the left-hand-side. The equality cannot hold, no matter how \(A, B\) are chosen. The second try fails.
    We will have to multiply by another \(t\) and try again.
    Now we try:
    \(Y = t^2 e^t(At + B) = (At^3 + Bt^2)e^t.\)
    Then \(Y' = (3At^2 + 2Bt)e^t + (At^3 + Bt^2)e^t,\)
    \(Y'' = (6At + 2B)e^t + 2(3At^2 + 2Bt)e^t + (At^3 + Bt^2)e^t.\) So,
    \(Y'' - 2Y' + Y\)
    \(= (6At + 2B)e^t + 2(3At^2 + 2Bt)e^t + (At^3 + Bt^2)e^t\)
    \(- 2((3At^2 + 2Bt)e^t + (At^3 + Bt^2)e^t) + (At^3 + Bt^2)e^t\)
    \(= (6At + 2B)e^t = te^t.\)
    This means \(6A = 1, 2B = 0 \implies A = \dfrac{1}{6}, B = 0 \implies Y = \dfrac{1}{6}t^3 e^t.\)
    The general solution:
    \(y = C_1 e^t + C_2 te^t + \dfrac{1}{6}t^3 e^t.\)
    How to find the final ansatz without trying
    Recall: In \(y'' - y' - 2y = e^{2t}\) of Example 1:
    1. The aux. eqn. is \(r^2 - r - 2 = 0 \implies r_1 = 2, r_2 = -1.\)
    2. \(g(t) = e^{2t}\) whose exponential coefficient is \(2,\) which is a root that appears once.
    3. We have seen that the first ansatz failed, the second succeeded.
    Recall: In \(y'' - 2y' + y = te^t\) of Example 2:
    1. The aux. eqn. is \(r^2 - 2r + 1 = 0 \implies r_1 = r_2 = 1.\)
    2. \(g(t) = e^t\) whose exponential coefficient is \(1,\) which is a root that appears twice.
    3. We have seen that the first and second ansatz failed, the third ansatz succeeded.
    Generally, for the ODE \(ay'' + by' + cy = g(t),\) where
    \(g(t) = e^{at} \cos \beta t \cdot (\text{polynomial}) + e^{at} \sin \beta t \cdot (\text{polynomial}),\)
    if the complex number \(\alpha + i\beta\) appears as a root for the auxiliary equation for \(m\) times, then the first \(m\) ansatz fail. We should modify the first ansatz by \(t^m.\)
    Remark: For convenience, we will call the complex number \(\alpha + i\beta\) the exponential coefficient of \(g(t).\)
    Remark: The exponential coefficient is a nonstandard terminology. In higher level mathematics, \(\alpha + i\beta\) is indeed an eigenvalue with respect to the differential operator. Its associated eigenfunction is \(e^{(\alpha + i\beta)t}\) with real part \(e^{at} \cos \beta t\) and imaginary part \(e^{at} \sin \beta t.\) But itâ€™s wrong to call these trigs eigenfunctions.
    Remark: Since the ODE is second-order, there are at most two roots for the auxiliary equation, the ansatz at most fails twice.
    Example 3: \(y'' + 4y = t \sin 2t.\)
    Aux. Eqn: \(r^2 + 4 = 0 \implies r^2 = -4 \implies r = \pm 2i.\)
    Complementary solution: \(y_c = C_1 \cos 2t + C_2 \sin 2t.\)
    The first ansatz: \(Y = (At + B) \cos 2t + (Ct + D) \sin 2t.\)
    Exponential coefficient: \(0 + 2i = 2i\) appears as a single root.
    First try fails. The final ansatz should be
    \(Y = t((At + B) \cos 2t + (Ct + D) \sin 2t)\)
    \(= (At^2 + Bt) \cos 2t + (Ct^2 + Dt) \sin 2t.\)
    \(Y' = (2At + B) \cos 2t - 2(At^2 + Bt) \sin 2t + (2Ct + D) \sin 2t + 2(Ct^2 + D) \cos 2t.\)
    To compute \(Y'',\) we may use the binomial theorem \((fg)'' = f''g + 2f'g' + fg''.\)
    Then
    \(Y'' = 2A \cos 2t + 2(2At + B) \cdot (-2 \sin 2t) + (At^2 + Bt) \cdot (-4 \cos 2t) + 2C \sin 2t + 2(2Ct + D) \cdot 2 \cos 2t + (Ct^2 + Dt) \cdot (-4 \sin 2t).\)
    \(Y'' + 4Y = 2A \cos 2t + 2(2At + B) \cdot (-2 \sin 2t) + 2C \sin 2t + 2(2Ct + D) \cdot 2 \cos 2t\)
    \(= \cos 2t (8Ct + 2A + 4D) - 2 \sin 2t (-8At - 4B + 2C).\)
    \(Y'' + 4Y\)
    \(= \cos 2t (8Ct + 2A + 4D) + \sin 2t (-8At - 4B + 2C)\)
    \(= t \sin 2t.\)
    Comparing the coefficients, we see that
    \(8C = 0, \quad 2A + 4D = 0, \quad -8A = 1, \quad -4B + 2C = 0\)
    \(\implies A = -\dfrac{1}{8}, B = C = 0, D = \dfrac{1}{16}.\)
    So \(Y = -\dfrac{1}{8}t^2 \cos 2t + \dfrac{1}{16}t \sin 2t.\) The general solution is
    \(y = C_1 \cos 2t + C_2 \sin 2t - \dfrac{1}{8}t^2 \cos 2t + \dfrac{1}{16}t \sin 2t.\)
    Example 4: \(y'' - 2y' + 2y = e^t \cos t.\)
    Aux. Eqn: \(r^2 - 2r + 2 = 0 \implies (r - 1)^2 = -1 \implies r = 1 \pm i.\)
    Comp. Soln: \(y_c = C_1 e^t \cos t + C_2 e^t \sin t.\)
    First ansatz: \(Y = Ae^t \cos t + Be^t \sin t.\)
    Exp. Coeff.: \(= 1 + i\) that appear as a single root. First try fails.
    Final ansatz: \(Y = Ate^t \cos t + Bte^t \sin t.\)
    To compute the derivatives, we may use the following rules:
    \((fg)' = f'g h + f g'h + f g h'.\)
    So
    \(Y' = A(e^t \cos t + te^t \cos t - te^t \sin t) + B(e^t \sin t + te^t \sin t + te^t \cos t)\)
    \(= ((A + B)t + A)e^t \cos t + ((B - A)t + B)e^t \sin t.\)
    \(Y'' = (A + B)e^t \cos t + ((A + B)t + A)e^t \cos t\)
    \(+ ((A + B)t + A)e^t \cdot (-\sin t) + (B - A)e^t \sin t.\)
    \(+ ((B - A)t + B)e^t \cos t.\)
    \(Y'' = (A + B)e^t \cos t + ((A + B)t + A)e^t \cos t\)
    \(+ ((A + B)t + A)e^t \cdot (-\sin t) + (B - A)e^t \sin t\)
    \(+ ((B - A)t + B)e^t \cos t\)
    \(= ((A + B) + (A + B)t + A + (B - A)t + B)e^t \cos t\)
    \(+ ((-(A + B)t - A + B - A)t + B)e^t \sin t\)
    \(= (2Bt + 2A + 2B)e^t \cos t + (-2At - 2A + 2B)e^t \sin t.\)
    So
    \(Y'' - 2Y' + 2Y\)
    \(= (2Bt + 2A + 2B - 2((A + B)t + A) + 2At)e^t \cos t\)
    \(+ (-2At - 2A + 2B - 2((B - A)t + B) + 2Bt)e^t \sin t\)
    \(= 2Be^t \cos t - 2Ae^t \sin t = e^t \cos t.\)
    \(\implies 2B = 1, -2A = 0 \implies A = 0, B = \dfrac{1}{2} \implies Y = \dfrac{1}{2}te^t \sin t.\)
    General solution:
    \(y = C_1 e^t \cos t + C_2 e^t \sin t + \dfrac{1}{2} te^t \sin t.\)
    Note: In the MIT Lecture 13 and 14, there exists a simpler way of computing \(Y'' - 2Y' + 2Y\text{,}\) using the exponential shift lemma of differential operators and complex functions. We shall not require that in our course.
    Sums of functions with different exp coeffs.
    We have seen how to solve \(ay'' + by' + cy = g(t)\) when
    \(g(t) = e^{at} \cos \beta t \cdot (\text{polynomial}) + e^{at} \sin \beta t \cdot (\text{polynomial}).\)
    What shall we do if \(g(t)\) is a sum of such functions but with different Î±â€™s and Î²â€™s? For example,
    \(g(t) = t^3 + e^t + \sin 2t + 2 \cos 2t.\)
    To handle such \(g(t),\) we need the following principle of superposition.
    4. Let \(Y_1\) be a particular solution of
    \(y'' + p(t)y' + q(t)y = g_1(t),\)
    and \(Y_2\) be a particular solution of
    \(y'' + p(t)y' + q(t)y = g_2(t).\)
    Then \(Y_1 + Y_2\) is a particular solution of
    \(y'' + p(t)y' + q(t)y = g_1(t) + g_2(t).\)
    Proof. The assumption implies that
    \(Y_1'' + p(t)Y_1' + q(t)Y_1 = g_1(t),\)
    \(Y_2'' + p(t)Y_2' + q(t)Y_2 = g_2(t).\)
    Adding them up, by linearity of differential operators, we have
    \((Y_1 + Y_2)'' + p(t)(Y_1 + Y_2)' + q(t)(Y_1 + Y_2) = g_1(t) + g_2(t).\)
    Thus \(Y_1 + Y_2\) satisfies the ODE
    \(y'' + p(t)y' + q(t)y = g_1(t) + g_2(t).\)
    The principle of superposition tells us that to handle such \(g(t):\)
    1. Break \(g(t) = g_1(t) + \cdots + g_m(t),\) according to the exponential coefficients.
    2. Find \(Y_1, \ldots, Y_m\) individually for \(g_1, \ldots, g_m,\) i.e., for \(i = 1, \ldots, m,\) find a particular \(Y_i\) for \(ay'' + by' + cy = g_i(t).\)
    3. Then \(Y = Y_1 + \cdots + Y_m\) is a particular solution.
    Example 5. \(y'' + y = t + t \sin t + 2 \cos t.\)
    Aux. Eqn: \(r^2 + 1 = 0 \implies r = \pm i.\)
    Comp. Soln: \(y_c = C_1 \cos t + C_2 \sin t.\)
    Now we regard \(g(t) = g_1(t) + g_2(t),\) where
    \(g_1(t) = t, \quad g_2(t) = t \sin t + 2 \cos t.\)
    (Note: Terms with the same exponential coefficient should be collected together to avoid unnecessary repetitions.)
    First we deal with \(g_1(t) = t.\) The first ansatz is \(Y_1 = At + B.\) The exponential coefficient is \(0\) that is not a root. First ansatz succeeds.
    \(Y_1' = A, \quad Y_1'' = 0. Y_1'' + Y_1 = At + B = g_1(t) = t\)
    \(\implies A = 1, B = 0 \implies Y_1 = t.\)
    The first ansatz is \(Y_2 = (Ct + D) \cos t + (Et + F) \sin t.\)
    The exponential coefficient is \(i\) that appears as a single root. First try fails. The final ansatz is
    \(Y_2 = (Ct^2 + Dt) \cos t + (Et^2 + Ft) \sin t.\)
    We compute the derivatives and find that
    \(Y_2'' + Y_2 = (-4Ct + 2E - 2D) \sin t + (4Et + 2F + 2C) \cos t.\)
    \(= t \sin t + 2 \cos t\)
    \(\implies -4C = 1, 2E - 2D = 0, 4E = 0, 2F + 2C = 2.\)
    \(\implies C = -\dfrac{1}{4}, D = E = 0, F = \dfrac{5}{4}.\)
    We found solutions
    \(Y_1 = t, \quad Y_2 = -\dfrac{1}{4}t^2 \cos t + \dfrac{5}{4}t \sin t.\)
    From the principle of superposition,
    \(Y = Y_1 + Y_2 = t - \dfrac{1}{4}t^2 \cos t + \dfrac{5}{4}t \sin t\)
    is a particular solution. The general solution is then
    \(y = C_1 \cos t + C_2 \sin t + t - \dfrac{1}{4}t^2 \cos t + \dfrac{5}{4}t \sin t.\)
