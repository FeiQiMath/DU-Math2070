    Section 1.7 Existence and Uniqueness of Solutions

    For the application of ODEs in science and engineering, when we modeled a problem into an initial value problem, normally we expect the solution to exist and be unique. However, this is not always the case.

    Subsection 1.7.1 Pathological IVP

    \begin{equation*} \dfrac{dy}{dx} = x\sqrt{y}, \, y(0) = 0 \end{equation*}
    This IVP has two solutions:
    \begin{equation*} y = \dfrac{1}{16}x^2 \text{ and }y = 0. \end{equation*}
    If a system is modeled by this IVP, then we cannot possibly predict how the system behaves. Such problem can also happen for linear ODeS.

    Example 1.7.1. \(ty' + (t - 1)y = -e^{-t}, \, y(0) = 1\).

    • Standard form:
    • Integrating factor:
    • General solution:
    • Plug in the initial value:
      \begin{equation*} y(0) = 1 \implies 1 + 0 = 1 \end{equation*}
      This means for every real number \(C\text{,}\) \(y(t) = e^{-t} + Cte^{-t}\) is a solution of the IVP. The IVP has infinitely many solutions.

    Example 1.7.2. \(ty' + (t - 1)y = -e^{-t}, \, y(0) = 0\).

    • We have obtained the general solution in Example 1.7.1: \(y(t) = e^{-t} + Cte^{-t}.\)
    • Plug in the initial value: \(y(0) = 0 \implies 1 + 0 = 0\text{,}\) impossible. In other words, The IVP has no solutions.
    Summary: A pathological IVP may have no solution / more than one solution / infinitely many solutions. To determine if an IVP is reasonably posed, we need the existence and uniqueness theorem.

    Subsection 1.7.2 Existence and Uniqueness Theorem: Linear version.

    Theorem 1.7.3. Existence and Uniqueness Theorem: Linear version..

    Consider the IVP
    \begin{equation*} y' + p(t)y = g(t), \, y(t_0) = y_0 \end{equation*}
    where the ODE is linear and in its standard form. Assume that:
    1. Both \(p(t)\) and \(g(t)\) are continuous on an open interval \((a,b).\)
    2. The open interval \((a,b)\) contains the initial time \(t_0.\)
    Then there exists a unique function \(y = y(t)\) over the interval \((a,b)\) that solves the IVP.

    Definition 1.7.4. Interval of Existence and Uniqueness.

    The interval of existence and uniqueness is an \((a,b)\) where the solution of the IVP exists uniquely. For brevity, we call it the interval of existence.
    Theorem 1.7.3 allows us to determine the interval of existence without solving it. To obtain the interval, we follow the following procedure.
    • Find out the standard form of the ODE \(y' + p(t)y = g(t).\) and specify the initial condition \(y(t_0) = y_0.\)
    • Find out the singular points, which are the points where \(p(t)\) and \(g(t)\) are not continuous. These singular points separate the real line into disjoint open intervals
    • If \(t_0\) is a singular point, we have a pathological IVP. If \(t_0\) is not a singular point, then the interval of existence is just the interval that contains \(t_0\) that are bounded by the singular points.

    Example 1.7.5. \(ty' + (t - 1)y = -e^{-t}, \, y(\ln 2) = \dfrac{1}{2}\).

    • Standard form:
      \begin{equation*} y' + \dfrac{t - 1}{t}y = -\dfrac{e^{-t}}{t}. \end{equation*}
    • Both \(\dfrac{t - 1}{t}\) and \(- \dfrac{e^{-t}}{t}\) blow up at \(0.\) So \(t = 0\) is a singular point.
    • (Missing a diagram here)
    Interval of existence = \((0, \infty).\)
    Remark. The solution of the IVP is \(y(t) = e^{-t}\) that is defined on \((-\infty, \infty).\) So if we talk about the interval of definition of the solution, it is \((-\infty, \infty).\) But this doesn’t change the fact that \(t = 0\) is a singular point. Generally, we should not be concerned about the solution outside the interval of existence, which is still \((0, \infty).\)

    Example 1.7.6. \((t - 3)y' + (\ln t)y = 2t, \, y(1) = 2.\).

    • Standard form:
      \begin{equation*} y' + \dfrac{\ln t}{t - 3}y = \dfrac{2t}{t - 3}. \end{equation*}
    • \(\dfrac{\ln t}{t - 3}\) is not continuous when \(t \lt 0\) and \(t = 3.\)
    • \(\dfrac{2t}{t - 3}\) is not continuous when \(t = 3.\)
    • (Missing a diagram here)
    Interval of existence = \((0, 3).\)

    Example 1.7.7. \(\sin(2t) y' + \tan(4t) y = \dfrac{1}{t}, \, y\left(\dfrac{\pi}{4}\right) = 0.\).

    • The standard form is
      \begin{equation*} y' + \dfrac{\tan 4t}{\sin 2t}y = \dfrac{1}{t \sin(2t)}. \end{equation*}
    • \(\dfrac{\tan 4t}{\sin 2t} = \dfrac{\sin 4t}{\cos 4t \sin 2t}\) is not continuous when \(\cos 4t = 0\) or \(\sin 2t = 0.\)
      Recall:
      \begin{align*} \amp \sin \alpha = 0 \text{ when } \alpha = k\pi, \, k = 0, \pm 1, \pm 2, \ldots\\ \amp \cos \alpha = 0 \text{ when } \alpha = k\pi + \dfrac{\pi}{2}, \, k = 0, \pm 1, \pm 2, \ldots \end{align*}
      So \(\dfrac{\tan 4t}{\sin 2t} = \dfrac{\sin 4t}{\cos 4t \sin 2t}\) is not continuous when \(\cos 4t = 0\) or \(\sin 2t = 0.\) Note that
      \begin{equation*} \cos 4t = 0 \text{ when }4t = k\pi + \dfrac{\pi}{2} \implies t = \dfrac{k\pi}{4} + \dfrac{\pi}{8}, k = 0, \pm 1, \pm 2, \ldots. \end{equation*}
      and
      \begin{equation*} \sin 2t = 0 \text{ when }2t = k\pi \implies t = \dfrac{k\pi}{2}, k = 0, \pm 1, \pm 2, \ldots. \end{equation*}
    • \(\dfrac{1}{t \sin 2t}\) is not continuous when \(t = 0\text{,}\) or \(t = \dfrac{k\pi}{2},\) where \(k = 0, \pm 1, \pm 2, \ldots.\)
    • So the singular points are \(t = \dfrac{k\pi}{4} + \dfrac{\pi}{8}, k = 0, \pm 1, \pm 2, \ldots\) and \(t = \dfrac{k\pi}{2}, k = 0, \pm 1, \pm 2, \ldots.\)
    Interval of existence = \((\dfrac{\pi}{8}, \dfrac{3\pi}{8}).\)
    Remark: Recall in Example 1.5.7, we studied the IVP
    \begin{equation*} y' + y = g(t), \, y(0) = 0, \, g(t) = \begin{cases} 1 \amp 0 \leq t \leq 1 \\ 0 \amp t \gt 1 \end{cases} \end{equation*}
    Since \(g(t)\) is not continuous at \(1\text{,}\) the IVP only has a unique solution on \([0, 1].\) Indeed, for every number \(C,\)
    \begin{equation*} y(t) = \begin{cases} 1 - e^{-t} \amp t \in [0, 1] \\ Ce^{-t} \amp t \in (1, \infty) \end{cases} \end{equation*}
    is a solution (the IVP has infinitely many solutions). Only with the continuity requirement did us obtain a unique solution.

    Subsection 1.7.3 Existence and Uniqueness Theorem: Nonlinear case

    Theorem 1.7.8. Existence and Uniqueness Theorem: Nonlinear case.

    Consider the IVP
    \begin{equation*} y' = f(t,y), \, y(t_0) = y_0. \end{equation*}
    Assume that:
    1. The function \(f(t,y)\) is continuous NEAR \((t_0,y_0),\) i.e., one can find small \(\epsilon_1, \epsilon_2 \gt 0,\) such that \(f(t,y)\) is continuous in the rectangular region \((t_0 - \epsilon_1, t_0 + \epsilon_1) \times (y_0 - \epsilon_2, y_0 + \epsilon_2).\)
    2. The function \(\dfrac{\partial f}{\partial y}(t,y)\) is continuous NEAR \((t_0,y_0).\)
    Then a unique solution \(y = y(t)\) exists NEAR \(t = t_0,\) i.e., there exists a small \(\epsilon \gt 0\) such that a solution exists on \((t_0 - \epsilon, t_0 + \epsilon).\)
    Likewise, any IVP that does not satisfy the conditions of the theorem is pathological.
    (Missing a diagram here.)
    Remarks:
    1. The theorem is not as strong as the linear version. It only concludes local existence. It does not tell us the interval of existence.
    2. Nevertheless, the theorem tells if an IVP is reasonably formulated.
    Example 1: \(y' = y^{\dfrac{1}{3}}, \, y(0) = 1.\)
    1. \(f(t,y) = 3\sqrt[3]{y}\) is continuous everywhere.
    2. \(\dfrac{\partial f}{\partial y}(t,y) = \dfrac{1}{3\sqrt[3]{y^2}}\) is continuous when \(y \neq 0.\)
    \(y(0) = 1 \implies (t_0, y_0) = (0, 1)\) (away from \(y = 0).\)
    (Missing a diagram here.)
    So the IVP has a unique solution near \(t = 0.\)
    Example 2: \(y' = y^{\dfrac{1}{3}}, \, y(1) = 0.\)
    1. \(f(t,y) = 3\sqrt[3]{y}\) is continuous everywhere.
    2. \(\dfrac{\partial f}{\partial y}(t,y) = \dfrac{1}{3\sqrt[3]{y^2}}\) is continuous when \(y \neq 0.\)
    \(y(1) = 0 \implies (t_0, y_0) = (1, 0)\) (on \(y = 0).\)
    (Missing a diagram here.)
    The IVP does not have a unique solution.
    Exercise: Check that \(y = 0, \, y = \sqrt[3]{\dfrac{8}{27}(x - 1)^3}\) and \(y = -\sqrt[3]{\dfrac{8}{27}(x - 1)^3}\) are all solutions of the IVP.
    Example 3. Find out all \((t_0, y_0)\text{,}\) such that the IVP
    \begin{equation*} y' = \sqrt{1 - t^2 - y^2}, \, y(t_0) = y_0 \end{equation*}
    is reasonably formulated.
    \(f(t,y) = \sqrt{1 - t^2 - y^2}\) is continuous when \(t^2 + y^2 \leq 1.\)
    \(\dfrac{\partial f}{\partial y}(t,y) = -\dfrac{y}{\sqrt{1 - t^2 - y^2}}\) is continuous when \(t^2 + y^2 \lt 1.\)
    (Missing a diagram here.)
    So \((t_0, y_0)\) should satisfy \(t_0^2 + y_0^2 \lt 1.\)
