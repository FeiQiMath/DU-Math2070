    Section 3.1 Laplace Transform and Examples

    Subsection 3.1.1 Integral Transform

    Definition 3.1.1.

    Let \(K(s, t)\) be a function with two real variables. Given a function \(f(t)\) defined on a set \(D\text{,}\) the following integral
    \begin{equation*} F(s) = \int_D K(s, t)f(t)dt \end{equation*}
    defines a new function \(F(s)\text{.}\) The map sending \(f(t)\) to \(F(s)\) is called an integral transform. The function \(K(s, t)\) is called the kernel of the integral transform.
    ðŸ”—
    ðŸ”—

    Remark 3.1.2.

    Note that when \(D\) is an infinite interval, the integral defining \(F(s)\) is improper and might not converge for all real numbers \(s\text{.}\) In this case, the domain of new function \(F(s)\text{,}\) i.e., the set of \(s\) such that the integral converges, is not expected to be \((-\infty, \infty).\)
    ðŸ”—
    ðŸ”—
    Integral transforms appears in many discussions of mathematics, physics, engineering, and computer science. The most important one is the Fourier transform, where
    \begin{equation*} D = (-\infty, \infty), K(s, t) = e^{-ist}. \end{equation*}
    In other words, Fourier transform sends \(f(t)\) to a complex function
    \begin{equation*} F(s) = \int_{-\infty}^{\infty} f(t)e^{-ist} dt \end{equation*}
    Note that when \(D\) is an infinite interval, the integral defining \(F(s)\) is improper and might not converge for all real numbers \(s\text{.}\) In this case, the domain of new function \(F(s)\text{,}\) i.e., the set of \(s\) such that the integral converges, is not expected to be \((-\infty, \infty).\)
    ðŸ”—
    Fourier transform is widely used, including but not limited to, partial differential equations, computer graphics, machine learning, etc. We will not require it in this course because it is necessary to know complex function.
    ðŸ”—
    In this course, we mainly focus on the Laplace transform.
    ðŸ”—

    Definition 3.1.3.

    The Laplace transform is the integral transform with
    \begin{equation*} D = [0, \infty), K(s, t) = e^{-st}. \end{equation*}
    In other words, Laplace transform sends
    \begin{equation*} f(t) \mapsto F(s) = \int_{0}^{\infty} f(t)e^{-st} dt \end{equation*}
    The function \(F(s)\) is often denoted by \(\mathcal{L}\{f(t)\}.\)
    ðŸ”—
    The domain of \(F(s)\) depends on \(f(t)\text{,}\) determined by the convergence of the improper integral, i.e.,
    \begin{equation*} \lim_{b \to \infty} \int_{0}^{b} f(t)e^{-st} dt \end{equation*}
    should exist.
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Subsection 3.1.2 Examples of Laplace transforms

    Example 3.1.4. Evaluate \(\mathcal{L}\{1\}\).

    \begin{align*} \mathcal{L}\{1\} = \ \amp \int_{0}^{\infty} e^{-st} dt = \left. \dfrac{e^{-st}}{-s} \right|_{0}^{\infty} \\ = \ \amp \left( \lim_{t \to \infty} \dfrac{e^{-st}}{-s} \right) - \frac{1}{-s}. \end{align*}
    ðŸ”—
    • If \(s \gt 0\text{,}\) then \(\lim_{t \to \infty} \dfrac{e^{-st}}{-s} = 0.\) The integral converges. In this case,
      \begin{equation*} F(s) = 0 - \frac{1}{-s} = \dfrac{1}{s}. \end{equation*}
      ðŸ”—
    • If \(s \lt 0\text{,}\) then \(\lim_{t \to \infty} \dfrac{e^{-st}}{-s} = +\infty.\) The integral diverges.
      ðŸ”—
    • If \(s = 0\text{,}\) we cannot use the previous formula because \(1/s\) makes no sense. But looking back to the integral, note that when \(s = 0, e^{-st} = 1.\) In this case,
      \begin{equation*} \int_{0}^{\infty} e^{-st} dt = \int_{0}^{\infty} 1 dt \end{equation*}
      also diverges.
      ðŸ”—
    Conclusion:
    \begin{equation*} \mathcal{L}\{1\} = \dfrac{1}{s}, \quad s \gt 0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 3.1.5. Evaluate \(\mathcal{L}\{t\}\).

    \begin{align*} \mathcal{L}\{t\} = \ \amp \int_{0}^{\infty} te^{-st} dt = -\dfrac{1}{s} te^{-st} \bigg|_{0}^{\infty} - \int_{0}^{\infty} 1 \cdot \left(-\dfrac{1}{s} e^{-st}\right) dt\\ = \ \amp -\dfrac{1}{s} \left( \lim_{t \to \infty} te^{-st} - 0 \right) + \dfrac{1}{s} \int_{0}^{\infty} e^{-st} dt \end{align*}
    We have seen that \(\int_{0}^{\infty} e^{-st} dt\) converges if and only if \(s \gt 0.\) In this case, using Lâ€™HÃ´pitalâ€™s rule,
    \begin{equation*} \lim_{t \to \infty} te^{-st} = \lim_{t \to \infty} \dfrac{t}{e^{st}} = 0. \end{equation*}
    so
    \begin{align*} \mathcal{L}\{t\} = \ \amp \int_{0}^{\infty} te^{-st} dt = -\dfrac{1}{s} te^{-st} \bigg|_{0}^{\infty} - \int_{0}^{\infty} 1 \cdot \left(-\dfrac{1}{s} e^{-st}\right) dt\\ = \ \amp -\dfrac{1}{s} \left( \lim_{t \to \infty} te^{-st} - 0 \right) + \dfrac{1}{s} \int_{0}^{\infty} e^{-st} dt\\ = \ \amp -\dfrac{1}{s} \cdot (0 - 0) + \dfrac{1}{s} \cdot \dfrac{1}{s} = \dfrac{1}{s^2}. \end{align*}
    ðŸ”—
    Conclusion:
    \begin{equation*} \mathcal{L}\{t\} = \dfrac{1}{s^2}, \quad s \gt 0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 3.1.6. Evaluate \(\mathcal{L}\{t^n\}\).

    \begin{align*} \mathcal{L}\{t^n\} = \ \amp \int_{0}^{\infty} t^n e^{-st} dt\\ = \ \amp -\dfrac{1}{s} t^n e^{-st} \bigg|_{0}^{\infty} - \int_{0}^{\infty} nt^{n-1} \left(-\dfrac{1}{s} e^{-st}\right) dt\\ = \ \amp -\dfrac{1}{s} \left( \lim_{t \to \infty} t^n e^{-st} - 0 \right) + \dfrac{n}{s} \int_{0}^{\infty} t^{n-1} e^{-st} dt \end{align*}
    Clearly, \(\lim_{t \to \infty} t^n e^{-st}\) exists if and only if \(s \gt 0\text{,}\) and the limit is 0. So we have obtained that for \(s \gt 0,\)
    \begin{equation*} \mathcal{L}\{t^n\} = -\frac{n}{s} \int_{0}^{\infty} t^{n-1} e^{-st} dt = -\frac{n}{s} \mathcal{L}\{t^{n-1}\}. \end{equation*}
    This defines a recursion of \(\mathcal{L}\{t^n\}\) that can be easily solved:
    \begin{align*} \mathcal{L}\{t^n\} = \ \amp \dfrac{n}{s} \mathcal{L}\{t^{n-1}\} = \dfrac{n}{s} \cdot \dfrac{n - 1}{s} \mathcal{L}\{t^{n-2}\}\\ = \ \amp \cdots = \dfrac{n}{s} \cdot \dfrac{n - 1}{s} \cdots \dfrac{2}{s} \cdot \dfrac{1}{s} \mathcal{L}\{1\}\\ = \ \amp \dfrac{n \cdot (n - 1) \cdots 2 \cdot 1}{s^{n + 1}} = \dfrac{n!}{s^{n + 1}}. \end{align*}
    ðŸ”—
    Conclusion:
    \begin{equation*} \mathcal{L}\{t^n\} = \dfrac{n!}{s^{n + 1}}, \quad s \gt 0. \end{equation*}
    For example,
    \begin{equation*} \mathcal{L}\{t^2\} = \dfrac{2!}{s^3}, \quad \mathcal{L}\{t^3\} = \dfrac{3!}{s^4}, \quad \mathcal{L}\{t^4\} = \dfrac{4!}{s^5}, \quad s \gt 0. \end{equation*}
    Note that the power of \(s\) in the denominator is 1 larger than the power of \(t.\)
    ðŸ”—
    ðŸ”—

    Example 3.1.7. Evaluate \(\mathcal{L}\{e^{-3t}\}\) and \(\mathcal{L}\{e^{5t}\}\).

    • By definition,
      \begin{align*} \mathcal{L}\{e^{-3t}\} = \ \amp \int_{0}^{\infty} e^{-3t} e^{-st} dt = \int_{0}^{\infty} e^{-(s + 3)t} dt\\ = \ \amp \frac{1}{-(s + 3)} e^{-(s + 3)t} \bigg|_{0}^{\infty} = \frac{1}{-(s + 3)} \left( \lim_{t \to \infty} e^{-(s + 3)t} - 1 \right) \end{align*}
      Clearly, \(\lim_{t \to \infty} e^{-(s + 3)t}\) exists when \(s + 3 \gt 0,\) i.e., \(s \gt -3.\) The limit is zero. So
      \begin{equation*} \mathcal{L}\{e^{-3t}\} = \frac{1}{-(s + 3)} (0 - 1) = \frac{1}{s + 5}, \quad s \gt -3. \end{equation*}
      ðŸ”—
    • By definition,
      \begin{align*} \mathcal{L}\{e^{5t}\} = \ \amp \int_{0}^{\infty} e^{5t} e^{-st} dt = \int_{0}^{\infty} e^{-(s - 5)t} dt\\ = \ \amp \frac{1}{-(s - 5)} e^{-(s - 5)t} \bigg|_{0}^{\infty} = \frac{1}{-(s - 5)} \left( \lim_{t \to \infty} e^{-(s - 5)t} - 1 \right) \end{align*}
      Clearly, \(\lim_{t \to \infty} e^{-(s - 5)t}\) exists when \(s - 5 \gt 0,\) i.e., \(s \gt 5.\) The limit is zero. So
      \begin{equation*} \mathcal{L}\{e^{5t}\} = \frac{1}{-(s - 5)} (0 - 1) = \frac{1}{s - 5}, \quad s \gt 5. \end{equation*}
      ðŸ”—
    ðŸ”—

    Example 3.1.8. Evaluate \(\mathcal{L}\{e^{at}\}\).

    By definition,
    \begin{align*} \mathcal{L}\{e^{at}\} = \ \amp \int_{0}^{\infty} e^{at} e^{-st} dt = \int_{0}^{\infty} e^{-(s - \alpha)t} dt\\ = \ \amp \frac{1}{-(s - \alpha)} e^{-(s - \alpha)t} \bigg|_{0}^{\infty} = \frac{1}{-(s - \alpha)} \left( \lim_{t \to \infty} e^{-(s - \alpha)t} - 1 \right) \end{align*}
    Clearly, \(\lim_{t \to \infty} e^{-(s - \alpha)t}\) exists when \(s - \alpha \gt 0,\) i.e., \(s \gt \alpha.\) The limit is zero. So
    \begin{equation*} \mathcal{L}\{e^{at}\} = \frac{1}{s - \alpha}, \quad s \gt \alpha. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 3.1.9. Evaluate \(\mathcal{L}\{\sin 2t\}\).

    \begin{align*} \mathcal{L}\{\sin 2t\} = \ \amp \int_{0}^{\infty} \sin 2t \, e^{-st} dt\\ = \ \amp -\dfrac{1}{s} \sin 2t \, e^{-st} \bigg|_{0}^{\infty} - \int_{0}^{\infty} 2 \cos 2t \left(-\dfrac{1}{s} e^{-st}\right) dt\\ = \ \amp -\dfrac{1}{s} \left( \lim_{t \to \infty} e^{-st} \sin 2t - 0 \right) + \frac{2}{s} \int_{0}^{\infty} \cos 2t \, e^{-st} dt \end{align*}
    Clearly, \(\lim_{t \to \infty} e^{-st} \sin 2t\) exists iff \(s \gt 0,\) and equals 0. So
    \begin{align*} \mathcal{L}\{\sin 2t\} =\ \amp -\frac{2}{s} \int_{0}^{\infty} \cos 2t \, e^{-st} dt\\ = \ \amp \frac{2}{s} \left( -\dfrac{1}{s} \cos 2t \, e^{-st} \bigg|_{0}^{\infty} - \int_{0}^{\infty} (-2 \sin 2t) \left(-\dfrac{1}{s} e^{-st}\right) dt \right)\\ = \ \amp -\frac{2}{s^2} \left( \lim_{t \to \infty} \cos 2t \, e^{-st} - \cos 0 \, e^0 \right) - \frac{4}{s^2} \int_{0}^{\infty} \sin 2t \, e^{-st} dt \end{align*}
    Clearly when \(s \gt 0,\) \(\lim_{t \to \infty} e^{-st} \cos 2t = 0.\) So
    \begin{align*} \mathcal{L}\{\sin 2t\} = \ \amp -\frac{2}{s^2} (0 - 1) - \frac{4}{s^2} \int_{0}^{\infty} \sin 2t \, e^{-st} dt\\ = \ \amp \frac{2}{s^2} - \frac{4}{s^2} \mathcal{L}\{\sin 2t\} \Rightarrow \left( 1 + \frac{4}{s^2} \right) \mathcal{L}\{\sin 2t\} = \frac{2}{s^2} \end{align*}
    So we conclude that
    \begin{equation*} \mathcal{L}\{\sin 2t\} = \frac{2}{s^2 + 4}, \quad s \gt 0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 3.1.10. Evaluate \(\mathcal{L}\{\sin \beta t\}\).

    With a similar procedure, we see that
    \begin{align*} \mathcal{L}\{\sin \beta t\} = \ \amp \frac{\beta}{s^2} - \frac{\beta^2}{s^2} \mathcal{L}\{\sin \beta t\} \Rightarrow \left( 1 + \frac{\beta^2}{s^2} \right) \mathcal{L}\{\sin \beta t\} = \frac{\beta}{s^2} \end{align*}
    So we conclude that
    \begin{equation*} \mathcal{L}\{\sin \beta t\} = \frac{\beta}{s^2 + \beta^2}, \quad s \gt 0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 3.1.11. Evaluate \(\mathcal{L}\{\cos \beta t\}\).

    With a similar procedure, we see that
    \begin{align*} \mathcal{L}\{\cos \beta t\} = \ \amp \frac{1}{s} - \frac{\beta^2}{s^2} \mathcal{L}\{\cos \beta t\} \Rightarrow \left( 1 + \frac{\beta^2}{s^2} \right) \mathcal{L}\{\cos \beta t\} = \frac{s}{s^2} \end{align*}
    So we conclude that
    \begin{equation*} \mathcal{L}\{\cos \beta t\} = \frac{s}{s^2 + \beta^2}, \quad s \gt 0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Theorem 3.1.12. Linearity of Laplace transform.

    For every real numbers \(\alpha, \beta,\)
    \begin{equation*} \mathcal{L}\{\alpha f(t) + \beta g(t)\} = \alpha \mathcal{L}\{f(t)\} + \beta \mathcal{L}\{g(t)\}. \end{equation*}
    ðŸ”—
    ðŸ”—

    Proof.

    It follows from the linearity of integrals.
    \begin{align*} \mathcal{L}\{\alpha f(t) + \beta g(t)\} = \ \amp \int_{0}^{\infty} (\alpha f(t) + \beta g(t)) e^{-st} dt\\ = \ \amp \alpha \int_{0}^{\infty} f(t)e^{-st} dt + \beta \int_{0}^{\infty} g(t)e^{-st} dt\\ = \ \amp \alpha \mathcal{L}\{f(t)\} + \beta \mathcal{L}\{g(t)\}. \end{align*}
    ðŸ”—
    ðŸ”—

    Definition 3.1.13. Hyperbolic functions.

    Recall that from the Eulerâ€™s formula, the usual sine and cosine functions may also be expressed as
    \begin{equation*} \sin t = \frac{e^{it} - e^{-it}}{2i}, \quad \cos t = \frac{e^{it} + e^{-it}}{2}. \end{equation*}
    ðŸ”—
    The hyperbolic functions are defined similarly:
    \begin{equation*} \sinh t = \frac{e^t - e^{-t}}{2}, \quad \cosh t = \frac{e^t + e^{-t}}{2}. \end{equation*}
    ðŸ”—
    These functions are widely used in engineering practices.
    ðŸ”—
    ðŸ”—
    The Laplace transform of hyperbolic functions can be obtained via linearity. For every \(k \gt 0:\)
    \begin{align*} \mathcal{L}\{\sinh kt\} = \ \amp \mathcal{L}\left\{ \frac{e^{kt} - e^{-kt}}{2} \right\} = \frac{1}{2} \mathcal{L}\{e^{kt}\} - \frac{1}{2} \mathcal{L}\{e^{-kt}\}\\ = \ \amp \frac{1}{2} \cdot \frac{1}{s - k} - \frac{1}{2} \cdot \frac{1}{s + k} = \frac{1}{2} \cdot \frac{s + k - (s - k)}{(s - k)(s + k)} = \frac{k}{s^2 - k^2}. \end{align*}
    \begin{align*} \mathcal{L}\{\cosh kt\} = \ \amp \mathcal{L}\left\{ \frac{e^{kt} + e^{-kt}}{2} \right\} = \frac{1}{2} \mathcal{L}\{e^{kt}\} + \frac{1}{2} \mathcal{L}\{e^{-kt}\}\\ = \ \amp \frac{1}{2} \cdot \frac{1}{s - k} + \frac{1}{2} \cdot \frac{1}{s + k} = \frac{1}{2} \cdot \frac{s + k + (s - k)}{(s - k)(s + k)} = \frac{s}{s^2 - k^2}. \end{align*}
    ðŸ”—
    ðŸ”—

    Subsection 3.1.3 Summary of basic formulas for Laplace transform

    • Power functions:
      \begin{equation*} \mathcal{L}\{1\} = \dfrac{1}{s}, \quad \mathcal{L}\{t\} = \frac{1}{s^2}, \quad \mathcal{L}\{t^n\} = \frac{n!}{s^{n + 1}}, \quad s \gt 0. \end{equation*}
      Then by linearity, we may obtain the Laplace transform of all polynomial functions, e.g.,
      \begin{equation*} \mathcal{L}\{1 + 5t + 3t^2\} = \mathcal{L}\{1\} + 5\mathcal{L}\{t\} + 3\mathcal{L}\{t^2\} = \dfrac{1}{s} + \frac{5}{s^2} + 3 \cdot \frac{2!}{s^3} = \dfrac{1}{s} + \frac{5}{s^2} + \frac{6}{s^3}. \end{equation*}
      ðŸ”—
    • Exponential functions: for every real number \(\alpha,\)
      \begin{equation*} \mathcal{L}\{e^{\alpha t}\} = \frac{1}{s - \alpha}, \quad s \gt \alpha. \end{equation*}
      ðŸ”—
    • Trigonometric functions: for every \(\beta \gt 0,\)
      \begin{equation*} \mathcal{L}\{\cos \beta t\} = \frac{s}{s^2 + \beta^2}, \quad \mathcal{L}\{\sin \beta t\} = \frac{\beta}{s^2 + \beta^2}, \quad s \gt 0. \end{equation*}
      ðŸ”—
    • Hyperbolic functions, for every \(k \gt 0,\)
      \begin{equation*} \mathcal{L}\{\cosh kt\} = \frac{s}{s^2 - k^2}, \quad \mathcal{L}\{\sinh kt\} = \frac{k}{s^2 - k^2}, \quad s \gt k. \end{equation*}

    Subsection 3.1.4 Existence of \(\mathcal{L}\{f(t)\}\) and its long term behavior

    Recall: A function is piecewise continuous on \([0, \infty),\) if in any subinterval \([a, b],\) the function is continuous everywhere except at finitely many removable or jump discontinuities.
    ðŸ”—

    Definition 3.1.14.

    A function is said to be of exponential order if there exist constants \(c, M \gt 0\) and \(T \gt 0,\) such that
    \begin{equation*} |f(t)| \leq Me^{ct} \text{ for all } t \gt T. \end{equation*}
    In other words, \(|f(t)|\) does not grow faster than a constant multiple of a simple exponential function \(e^{ct}.\)
    ðŸ”—
    ðŸ”—
    Polynomial functions, simple exponential function \(e^{at},\) cosine and sine functions are of exponential order.
    ðŸ”—
    The function \(f(t) = e^{t^2}\) is not of exponential order, since for every possible \(c \gt 0,\)
    \begin{equation*} \left| \frac{e^{t^2}}{e^{ct}} \right| = e^{t^2 - ct} = e^{t^2(t - c)} \to \infty \end{equation*}
    So \(e^{t^2}\) grows faster than any multiple of \(e^{ct}.\)
    ðŸ”—

    Theorem 3.1.15. Sufficient conditions for existence of Laplace transform.

    If \(f\) is piecewise continuous on \([0, \infty)\) and of exponential order, then \(\mathcal{L}\{f(t)\}\) exists for \(s \gt c.\)
    ðŸ”—
    ðŸ”—

    Proof.

    Since \(f\) is of exponential order, there exist \(c \gt 0, M \gt 0, T \gt 0,\) such that whenever \(t \gt T, |f(t)| \leq Me^{ct}.\) Using the number \(T,\) we rewrite
    \begin{equation*} \mathcal{L}\{f(t)\} = \int_{0}^{T} e^{-st} f(t) dt + \int_{T}^{\infty} e^{-st} f(t) dt = I_1 + I_2 \end{equation*}
    Since \(f\) is piecewise continuous, \(I_1\) exists.
    ðŸ”—
    For \(I_2,\) we use the inequality \(|f(t)| \leq Me^{ct}\) to show that
    \begin{equation*} \int_{T}^{\infty} |e^{-st} f(t)| dt = \int_{T}^{\infty} e^{-st} |f(t)| dt \leq \int_{T}^{\infty} e^{-st} e^{ct} dt \end{equation*}
    Right-hand side is precisely \(\mathcal{L}\{e^{ct}\}\) that exists when \(s \gt c.\) In other words, \(I_2\) converges absolutely when \(s \gt c.\)
    ðŸ”—
    ðŸ”—

    Remark 3.1.16.

    These two conditions are sufficient but not necessary. In other words, \(\mathcal{L}\{f(t)\}\) exists does not mean that \(f(t)\) should satisfy these two conditions.
    ðŸ”—
    For example, the function \(t^{-\frac{1}{2}}\) is not piecewise continuous. The function \(2te^{t^2} \cos e^{t^2}\) is not of exponential order. But it is not difficult to see that both \(\mathcal{L}\{t^{-\frac{1}{2}}\}\) and \(\mathcal{L}\{2te^{t^2} \cos e^{t^2}\}\) exist.
    ðŸ”—
    ðŸ”—

    Theorem 3.1.17. Long term behavior of \(\mathcal{L}\{f(t)\}\).

    If \(f\) is piecewise continuous on \([0, \infty)\) and of exponential order, then the function \(F(s) = \mathcal{L}\{f(t)\}\) satisfies \(\lim_{s \to \infty} F(s) = 0.\)
    ðŸ”—
    ðŸ”—

    Proof.

    Take \(c, M, T \gt 0\) as in the previous proof. Since \(f\) is piecewise continuous on \([0, T],\) \(f(x)\) is bounded. By adjusting \(M,\) we may assume that \(|f(t)| \leq Me^{ct}\) on \([0, \infty).\) Then we estimate that
    \begin{equation*} 0 \leq |F(s)| \leq \int_{0}^{\infty} e^{-st} |f(t)| dt \leq \int_{0}^{\infty} e^{-st} e^{ct} dt = \int_{0}^{\infty} e^{-(s - c)t} dt = \frac{1}{s - c} \to 0, \text{ as } s \to \infty. \end{equation*}
    The conclusion then follows from the squeeze theorem.
