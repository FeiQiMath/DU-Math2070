    Section 2.7 Method of Undetermined Coefficients - First Ansatz

    Subsection 2.7.1 Basic Types

    We use examples to study the method of undetermined coefficients.

    Example 2.7.1. \(y'' - 2y' - 3y = 3\).

    Idea: RHS is a constant. We try a constant function.
    • Complementary solution: \(y_c = C_1e^{3t} + C_2e^{-t}.\)
    • Set \(Y = A.\) Then \(Y' = 0, Y'' = 0.\)
    • Plug the derivatives into the LHS:
      \begin{equation*} Y'' - 2Y' - 3Y = 0 - 0 - 3A = 3 \end{equation*}
      Comparing both sides, we see that
      \begin{equation*} A = -1 \implies Y = -1. \end{equation*}
    Conclusion: The general solution is
    \begin{equation*} y = C_1e^{3t} + C_2e^{-t} - 1. \end{equation*}

    Example 2.7.2. \(y'' - y' - 2y = t^2 + 1\).

    Idea: RHS is a polynomial. We try a polynomial function.
    • Complementary solution: \(y_c = C_1e^{2t} + C_2e^{-t}.\)
    • Set \(Y = At^2 + Bt + C.\) Then
      \begin{equation*} Y' = 2At + B, Y'' = 2A. \end{equation*}
    • Plug the derivatives into the LHS:
      \begin{align*} \amp Y'' - Y' - 2Y \\ = \amp 2A - (2At + B) - 2(At^2 + Bt + C) \\ = \amp -2At^2 + (-2A - 2B)t + (2A - B - 2C) \\ = \amp t^2 + 1 \end{align*}
      Comparing the coefficients of powers of \(t\) on both sides, we get
      \begin{align*} \amp -2A = 1, -2A - 2B = 0, 2A - B - 2C = 1\\ \implies \amp A = -\dfrac{1}{2}, B = \dfrac{1}{2}, C = -\dfrac{5}{4} \\ \implies \amp Y = -\dfrac{1}{2}t^2 + \dfrac{1}{2}t - \dfrac{5}{4}. \end{align*}
    Conclusion: The general solution is
    \begin{equation*} y = C_1e^{2t} + C_2e^{-t} - \dfrac{1}{2}t^2 + \dfrac{1}{2}t - \dfrac{5}{4}. \end{equation*}

    Remark 2.7.3.

    1. Generally, if the RHS is a polynomial of degree \(n\text{,}\) we set \(Y\) as a generic polynomial of degree \(n.\)
    2. Even if \(g(t) = t^n\text{,}\) when setting \(Y\text{,}\) we should start from the highest degree of \(g(t)\) and go all the way down to the constant term.

    Example 2.7.4. \(y'' - y' - 2y = t^3\).

    • Complementary solution: \(y_c = C_1e^{2t} + C_2e^{-t}.\)
    • Set \(Y = At^3 + Bt^2 + Ct + D.\) Then
      \begin{equation*} Y' = 3At^2 + 2Bt + C, Y'' = 6At + 2B. \end{equation*}
    • Plug the derivatives into the LHS:
      \begin{align*} \amp Y'' - Y'- 2Y \\ = \amp 6At + 2B - (3At^2 + 2Bt + C) - 2(At^3 + Bt^2 + Ct + D)\\ = \amp -2At^3 + (-2B - 3A)t^2 + (6A - 2B - 2C)t + (2B - C - 2D) \\ = \amp t^3 \end{align*}
      Comparing the coefficients of powers of \(t\text{,}\) we get
      \begin{align*} \amp -2A = 1, -2B - 3A = 0, \\ \amp 6A - 2B - 2C = 0, 2B - C - 2D = 0\\ \implies \amp A = -\dfrac{1}{2}, B = -\dfrac{3}{4}, C = \dfrac{9}{4}, D = -\dfrac{3}{8}.\\ \implies \amp Y = -\dfrac{1}{2}t^3 + \dfrac{3}{4}t^2 + \dfrac{9}{4}t - \dfrac{3}{8}. \end{align*}
    General solution:
    \begin{equation*} y = C_1e^{2t} + C_2e^{-t} - \dfrac{1}{2}t^3 + \dfrac{3}{4}t^2 + \dfrac{9}{4}t - \dfrac{3}{8}. \end{equation*}

    Example 2.7.5. \(y'' - y' - 2y = e^{3t}\).

    Idea: RHS is an exponential function. We try an exponential function with the same exponential coefficient.
    • Complementary solution: \(y_c = C_1e^{2t} + C_2e^{-t}.\)
    • Set \(Y = Ae^{3t}.\) Then
      \begin{equation*} Y' = 3Ae^{3t}, Y'' = 9Ae^{3t}. \end{equation*}
    • Plug the derivatives into the LHS:
      \begin{align*} \amp Y'' - Y' - 2Y \\ = \amp 9Ae^{3t} - 3Ae^{3t} - 2Ae^{3t} = 4Ae^{3t} = e^{3t} \end{align*}
      Comparing the coefficients of \(e^{3t}\) on both sides, we get
      \begin{equation*} 4A = 1 \implies A = \dfrac{1}{4} \implies Y = \dfrac{1}{4}e^{3t}. \end{equation*}
    General solution:
    \begin{equation*} y = C_1e^{2t} + C_2e^{-t} + \dfrac{1}{4}e^{3t}. \end{equation*}

    Remark 2.7.6.

    If we choose \(e^{\alpha t}\) for \(\alpha \neq 3,\) it won’t work.

    Example 2.7.7. \(y'' - 2y' + y = 3 \sin 3t\).

    Idea: RHS is a trigonometric function. We try a trigonometric function with the same trigonometric coefficient.
    • Complementary solution: \(y_c = C_1e^t + C_2te^t.\)
    • Set \(Y = A \sin 3t + B \cos 3t.\) Note that both sin and cos should appear. Then
      \begin{equation*} Y' = 3A \cos 3t - 3B \sin 3t, Y'' = -9A \sin 3t - 9B \cos 3t. \end{equation*}
    • Plug the derivatives into the LHS:
      \begin{align*} \amp Y'' - 2Y' + Y \\ = \amp (-9A \sin 3t - 9B \cos 3t) - 2(3A \cos 3t - 3B \sin 3t) \\ \amp + (A \sin 3t + B \cos 3t)\\ = \amp (-8A + 6B) \sin 3t + (-8B - 6A) \cos 3t = 3 \sin 3t. \end{align*}
      Comparing the coefficients of \(\sin 3t\) and \(\cos 3t\) on both sides, we get
      \begin{align*} \amp -8A + 6B = 3, -8B - 6A = 0\\ \implies \amp A = -\dfrac{6}{25}, B = -\dfrac{9}{50} \\ \implies \amp Y = -\dfrac{6}{25} \sin 3t - \dfrac{9}{50} \cos 3t. \end{align*}
    General solution:
    \begin{equation*} y = C_1e^t + C_2te^t - \dfrac{6}{25} \sin 3t + \dfrac{9}{50} \cos 3t. \end{equation*}

    Remark 2.7.8.

    1. This works only for sine and cosine functions, fails for other trigonometric functions.
    2. Note that both sine and cosine functions should appear in the ansatz. This is because the sine function or cosine function alone is not closed under the differentiation.
    3. If we try \(A \sin \alpha t + B \cos \alpha t\) for \(\alpha \neq 3,\) the method won’t work.

    Remark 2.7.9. (Important!).

    We have seen that for \(ay'' + by' + cy = g(t).\)
    1. If \(g(t) = 3\) (or \(g(t) = -10\)), we should set the ansatz as \(Y = A.\)
    2. If \(g(t) = t^2 + 1\) (or \(g(t) = 6t^2\text{,}\) or \(g(t) = -2t^2 + 4t - 3\)), we should set the ansatz as \(Y = At^2 + Bt + C.\)
    3. If \(g(t) = e^{3t}\) (or \(g(t) = -2e^{3t}\)), we should set the ansatz as \(Y = Ae^{3t}.\)
    4. If \(g(t) = 3 \sin 3t\) (or \(g(t) = 6 \cos 3t\text{,}\) or \(g(t) = 18 \cos 3t - 27 \sin 3t\)), we should set the ansatz as \(Y = A \cos 3t + B \sin 3t.\)
    The method works because:
    • Derivatives of polynomial functions are polynomials (up to the same degree).
    • Derivatives of exponential functions are exponential functions (with the same exponential coefficient).
    • Derivatives of sine and cosine functions are sine and cosine functions (with the same trigonometric coefficient).
    For functions without this property, the method fails. We will have no choice but to use the integration formula.
    We call the polynomials, exponentials, sine and cosine functions to be basic types.

    Subsection 2.7.2 Products of Basic Types

    Observe that products of basic type functions satisfy the same property as in Remark 2.7.9. Therefore, we may continue the list of \(g(t)\) with products of basic types.
    1. If \(g(t) = 4t^3e^{3t}\) (or \(g(t) = (t^3 + 2t)e^{3t}\)), we should set the ansatz as
      \begin{equation*} Y = (At^3 + Bt^2 + Ct + D)e^{3t}. \end{equation*}
      Remark: We need an exponential function with the same coefficient. Then we pair it with a generic polynomial.
    2. If \(g(t) = (t^2 + 1) \sin 3t\) (or \(g(t) = 2t^2 \cos 3t,\) or \(g(t) = 2t \cos 3t + (t^2 + 1) \sin 3t\)), we should set the ansatz as
      \begin{equation*} Y = (At^2 + Bt + C) \sin 3t + (Dt^2 + Et + F) \cos 3t. \end{equation*}
      Remark: We need both sine and cosine with the same coefficient. We pair both sine and cosine with generic polynomials of the same degrees.
      A common mistake to avoid is to set \(Y = (At^2 + Bt + C)(D \cos 3t + E \sin 3t). \) This ansatz basically requires the coefficient of \(\sin 3t\) and \(\cos 3t\) to be proportional, which is too special.
    3. If \(g(t) = e^{3t} \cos 4t\) (or \(g(t) = -e^{3t} \sin 4t,\) or \(g(t) = 2e^{3t} \cos 4t - 6e^{3t} \sin 4t\)), we should set the ansatz as
      \begin{equation*} Y = Ae^{3t} \cos 4t + Be^{3t} \sin 4t. \end{equation*}
      Remark: We should understand \(e^{\alpha t} \cos \beta t\) (or \(e^{\alpha t} \sin \beta t,\) or both) as the real parts (or imaginary parts, or both) of \(e^{(\alpha + \beta i)t}\) and \(e^{(\alpha - \beta i)t}.\) We need both \(e^{\alpha t} \cos \beta t\) and \(e^{\alpha t} \sin \beta t.\)
    4. If \(g(t) = 3te^{3t} \cos 4t\) (or \(g(t) = 6te^{3t} \sin 4t,\) or \(g(t) = 18te^{3t} \cos 4t - 7e^{3t} \sin 4t\)), we should set the ansatz as
      \begin{equation*} Y = (At + B)e^{3t} \cos 4t + (Ct + D)e^{3t} \sin 4t. \end{equation*}
      Remark: We need both \(e^{\alpha t} \cos \beta t\) and \(e^{\alpha t} \sin \beta t.\) Then we pair both of them with generic polynomials of the same degree.

    Example 2.7.10. \(y'' + 4y = t e^t \cos 2t.\).

    • Complementary solution: \(y = C_1 \cos 2t + C_2 \sin 2t.\)
    • Set
      \begin{equation*} Y = (At + B)e^t \cos 2t + (Ct + D)e^t \sin 2t. \end{equation*}
      Then
      \begin{align*} Y' = \amp Ae^t \cos 2t + (At + B)e^t \cos 2t - 2(At + B)e^t \sin 2t \\ \amp + Ce^t \sin 2t + (Ct + D)e^t \sin 2t\\ = \amp [(A + 2C)t + B + A + 2D]e^t \cos 2t \\ \amp + [(C - 2A)t + C + D - 2B]e^t \sin 2t.\\ Y'' = \amp (A + 2C)e^t \cos 2t + [(A + 2C)t + B + A + 2D]e^t \cos 2t \\ \amp - 2[(A + 2C)t + B + A + 2D]e^t \sin 2t\\ \amp + (C - 2A)e^t \sin 2t + [(A + 2C)t + B + A + 2D]e^t \sin 2t\\ \amp + 2[(A + 2C)t + B + A + 2D]e^t \cos 2t.\\ = \amp [(-3A + 4C)t + 2A - 3B + 4C + 4D]e^t \cos 2t \\ \amp + [(-4A - 3C)t - 4A - 4B + 2C + D]e^t \sin 2t \end{align*}
    • Plug the derivatives into the LHS:
      \begin{align*} \amp Y'' + 4Y \\ = \amp[(-3A + 4C)t + 2A - 3B + 4C + 4D]e^t \cos 2t \\ \amp + [(-4A - 3C)t - 4A - 4B + 2C + D]e^t \sin 2t\\ \amp + 4(At + B)e^t \cos 2t + 4(Ct + D)e^t \sin 2t\\ = \amp [(A + 4C)t + 2A + B + 4C + 4D]e^t \cos 2t \\ \amp + [(-4A + C)t - 4A - 4B + 2C + D]e^t \sin 2t\\ = \amp te^t \cos 2t. \end{align*}
      Thus, we obtain four equations concerning four variables:
      \begin{gather*} A + 4C = 1, \quad 2A + B + 4C + 4D = 0,\\ -4A + C = 0, \quad -4A - 4B + 2C + D = 0. \end{gather*}
      Solving the system, we obtain
      \begin{align*} \amp A = \dfrac{1}{17}, \quad B = -\dfrac{2}{289}, \quad C = \dfrac{4}{17}, \quad D = -\dfrac{76}{289}.\\ \implies \amp Y = \left( \dfrac{1}{17} t - \dfrac{2}{289} \right)e^t \cos 2t + \left( \dfrac{4}{17} t - \dfrac{76}{289} \right)e^t \sin 2t. \end{align*}
    Conclusion: The general solution is
    \begin{align*} y = \amp C_1 \cos 2t + C_2 \sin 2t \\ \amp + \left( \dfrac{1}{17} t - \dfrac{2}{289} \right)e^t \cos 2t + \left( \dfrac{4}{17} t - \dfrac{76}{289} \right)e^t \sin 2t. \end{align*}

    Subsection 2.7.3 Summary of the first ansatz

    All the ansatz from 1-8 can be summarized as follows:
    For the ODE \(ay'' + by' + cy = g(t),\) with
    \begin{equation*} g(t) = e^{\alpha t} \cos(\beta t) (\text{polynomial}) + e^{\alpha t} \sin(\beta t) (\text{polynomial}), \end{equation*}
    the first ansatz should be set as
    \begin{align*} Y = \amp e^{\alpha t} \cos(\beta t) (A_n t^n + A_{n-1} t^{n-1} + \cdots + A_1 t + A_0) \\ \amp + e^{\alpha t} \sin(\beta t) (B_n t^n + B_{n-1} t^{n-1} + \cdots + B_1 t + B_0), \end{align*}
    where \(n\) is the largest degree of the polynomials appeared in \(g(t).\)

    Remark 2.7.11. (Important!).

    The first ansatz depends only on \(g(t),\) irrelevant of the left-hand-side. We call it the first ansatz because sometimes it might fail and might need to be further modified, as shall be seen in the next section.
