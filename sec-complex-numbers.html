    Section 2.3 Complex Numbers

    Subsection 2.3.1 Multiplication

    A complex number is of the form \(a + bi\text{,}\) where \(a, b\) are real numbers, and \(i = \sqrt{-1}.\) Note that \(i^2 = -1.\) \(a\) is called the real part, \(b\) is called the imaginary part.
    ðŸ”—
    Multiplication:
    \begin{equation*} (a + bi)(c + di) = ac + bci + adi + bd i^2 = ac - bd + (ad + bc)i. \end{equation*}
    ðŸ”—
    Examples:
    ðŸ”—
    • \(\displaystyle (1 + i)(2 - i) = 2 + 1 + (2i - i) = 3 + i.\)
      ðŸ”—
    • \(\displaystyle i(3 + i) = 3i - 1 = -1 + 3i.\)
      ðŸ”—
    • \(\displaystyle (\cos \theta + i \sin \theta)(1 - 2i) = \cos \theta + 2 \sin \theta + i(\sin \theta - 2 \cos \theta).\)

    Subsection 2.3.3 Euler Formula

    The famous Eulerâ€™s formula reads
    \begin{equation*} e^{i\theta} = \cos \theta + i \sin \theta \end{equation*}
    More generally,
    \begin{equation*} re^{i\theta} = r \cos \theta + i r \sin \theta, \, r \geq 0 \end{equation*}
    Indeed, this formula is nothing more than the polar-coordinate transformation
    \begin{equation*} (r, \theta) \leftrightarrow (r \cos \theta, r \sin \theta) \end{equation*}
    Given a complex number \(a + bi\text{,}\) the number \(r\text{,}\) called the amplitude, is precisely the distance between \(a + bi\) and the origin,
    \begin{equation*} r = \sqrt{a^2 + b^2} \end{equation*}
    The angle \(\theta\text{,}\) called the phase, is precisely the angle formed from the x-axis to the line connecting the origin and \(a + bi.\) In formulas,
    \begin{equation*} \theta = \begin{cases} \arctan\left(\frac{b}{a}\right) \amp \text{ if } (a, b) \text{ is in the first and fourth quadrant.} \\ \arctan\left(\frac{b}{a}\right) + \pi \amp \text{ if } (a, b) \text{ is in the second and third quadrant.} \end{cases} \end{equation*}
    ðŸ”—
    ðŸ”—
    Examples: \(1 + i = \sqrt{2} e^{i\frac{\pi}{4}}, \, 1 - i = \sqrt{2} e^{-i\frac{\pi}{4}} = \sqrt{2} e^{i\frac{7\pi}{4}}.\)
    ðŸ”—

    ðŸ”—

    Subsection 2.3.4 Geometric Interpretation of Multiplication

    Consider two complex numbers
    \begin{equation*} r_1 e^{i\theta_1} = r_1(\cos \theta_1 + i \sin \theta_1), \, r_2 e^{i\theta_2} = r_2(\cos \theta_2 + i \sin \theta_2). \end{equation*}
    Then multiplying these two complex numbers ends up with
    \begin{equation*} r_1 r_2 e^{i(\theta_1 + \theta_2)} = r_1 r_2 (\cos(\theta_1 + \theta_2) + i \sin(\theta_1 + \theta_2)). \end{equation*}
    ðŸ”—
    Proof:
    \begin{align*} \amp r_1(\cos \theta_1 + i \sin \theta_1) \cdot r_2(\cos \theta_2 + i \sin \theta_2)\\ = \amp r_1 r_2 \left( \cos \theta_2 \cos \theta_1 - \sin \theta_1 \sin \theta_2 \right)\\ = \amp r_1 r_2 \left( \cos \theta_2 \cos \theta_1 - \sin \theta_1 \sin \theta_2 \right)\\ \amp + i r_1 r_2 \left( \sin \theta_1 \cos \theta_2 + \sin \theta_2 \cos \theta_1 \right)\\ = \amp r_1 r_2 \left( +i(\sin \theta_1 \cos \theta_2 + \sin \theta_2 \cos \theta_1) \right)\\ = \amp r_1 r_2 (\cos(\theta_1 + \theta_2) + i \sin(\theta_1 + \theta_2)). \end{align*}
    Here we used the fact that
    \begin{equation*} \cos(\theta_1 + \theta_2) = \cos \theta_1 \cos \theta_2 - \sin \theta_1 \sin \theta_2, \end{equation*}
    and
    \begin{equation*} \sin(\theta_1 + \theta_2) = \sin \theta_1 \cos \theta_2 + \sin \theta_2 \cos \theta_1. \end{equation*}
    ðŸ”—
    Geometrically, this means that the amplitude of the product is the product of amplitudes, and phase of the product is the sum of the phases.
    ðŸ”—
    Example: \((1 + i)(1 - i) = \sqrt{2} e^{i\frac{\pi}{4}} \cdot \sqrt{2} e^{i\frac{3\pi}{4}} = 2 e^{i\pi} = -2.\)

    Subsection 2.3.5 Powers and Roots

    To square the complex number
    \begin{equation*} re^{i\theta} = r(\cos \theta + i \sin \theta), \end{equation*}
    using the previous description of multiplications, we obtain
    \begin{equation*} r^2 e^{i \cdot 2\theta} = r^2(\cos 2\theta + i \sin 2\theta). \end{equation*}
    Generally, for an integer \(n\text{,}\) the \(n\)-th power of \(re^{i\theta}\) is simply
    \begin{equation*} r^n e^{i n \theta} = r^n(\cos n\theta + i \sin n\theta). \end{equation*}
    ðŸ”—
    The roots, on the other hand, are more complicated. Generally, the \(n\)-th root \(\sqrt[n]{z}\) has \(n\) different candidates.
    ðŸ”—
    Let \(z = Re^{i\theta}, \, w = \sqrt[n]{z}.\) If
    \begin{equation*} w = re^{i\alpha}, \end{equation*}
    then
    \begin{equation*} r^n e^{i n \alpha} = Re^{i\theta}. \end{equation*}
    While the amplitude \(r = \sqrt[n]{R}\) is unique, the phase \(\alpha\) is not unique.
    ðŸ”—
    Indeed,
    \begin{equation*} e^{i n \alpha} = e^{i \theta} \text{ means } n\alpha = \theta + 2k\pi, \, k = 0, \pm 1, \pm 2, \ldots. \end{equation*}
    ðŸ”—
    This is to say that
    \begin{equation*} \alpha = \frac{\theta}{n} + \frac{2k\pi}{n}, \, k = 0, \pm 1, \pm 2, \ldots \end{equation*}
    Correspondingly, for \(k = 0, 1, \ldots, n - 1,\)
    \begin{equation*} \sqrt[n]{R} e^{i \frac{\theta}{n}}, \, \sqrt[n]{R} e^{i \left(\frac{\theta}{n} + \frac{2\pi k}{n}\right)}, \ldots, \, \sqrt[n]{R} e^{i \left(\frac{\theta}{n} + \frac{2(n-1)\pi}{n}\right)} \end{equation*}
    are \(n\) candidates of roots for \(\sqrt[n]{z}.\)
    ðŸ”—
    Example:
    \begin{align*} \sqrt[3]{1} = \amp \left(e^{i(0 + 2k\pi)}\right)^{\frac{1}{3}} = e^{i\frac{2k\pi}{3}}.\\ = \amp \begin{cases} e^{i0} = 1 \\ e^{i\frac{2\pi}{3}} = -\frac{1}{2} + \frac{\sqrt{3}}{2}i \\ e^{i\frac{4\pi}{3}} = -\frac{1}{2} - \frac{\sqrt{3}}{2}i \end{cases} \end{align*}
    ðŸ”—
    ðŸ”—
    Example:
    \begin{align*} \sqrt[4]{-4} = \amp \left(4 e^{i(\pi + 2k\pi)}\right)^{\frac{1}{4}} = \sqrt{2} e^{i\left(\frac{\pi}{4} + \frac{k\pi}{2}\right)}.\\ = \amp \begin{cases} \sqrt{2} e^{i\frac{\pi}{4}} \\ \sqrt{2} e^{i\frac{3\pi}{4}} \\ \sqrt{2} e^{i\frac{5\pi}{4}} \\ \sqrt{2} e^{i\frac{7\pi}{4}} \end{cases} \end{align*}
    may possibly be
    \begin{equation*} \sqrt{2} e^{i\frac{\pi}{4}}, \, \sqrt{2} e^{i\frac{3\pi}{4}}, \, \sqrt{2} e^{i\frac{5\pi}{4}}, \, \sqrt{2} e^{i\frac{7\pi}{4}}, \end{equation*}
    i.e.
    \begin{equation*} 1 + i, \, -1 + i, \, -1 - i, \, 1 - i. \end{equation*}
