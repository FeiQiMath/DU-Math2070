    Section 2.1 Homework 1

    1. Draw the direction field for the following ODEs.
      1. \begin{equation*} y' = 4 - y \end{equation*}
      2. \begin{equation*} y' = y + 2 \end{equation*}
      3. \begin{equation*} y' = (y+1)(y-2) \end{equation*}
    2. Classify the following differential equations by order and linearity.
      1. \begin{equation*} y^{(100)} + y' = 6 \end{equation*}
        Solution.
        The highest order of the derivative is 100. So the order is 100.
        Set \(F(t,y, ..., y^{(100)})=y^{(100)} + y' + 6\text{.}\) Then clearly,
        \begin{equation*} \frac{\partial F}{\partial y^{(i)}} = 0 \text{ if } i \neq 1 \text{ or } i \neq 100, \end{equation*}
        and
        \begin{equation*} \frac{\partial F}{\partial y^{'}} = \frac{\partial F}{\partial y^{(100)}}=1, \end{equation*}
        all of which are independent of \(y\) and its derivatives. So the ODE is linear.
      2. \begin{equation*} y''' + t^2 y'' + t^4 y' - (\cos t) y = \sqrt{t} \end{equation*}
        Solution.
        The highest order of the derivative is 3. So the order is 3.
        Set \(F(t,y, y', y'', y''')=y''' + t^2 y + t^4 y' - (\cos t)y - \sqrt{t}\text{.}\) Then clearly,
        \begin{equation*} \frac{\partial F}{\partial y} = - \cos t, \frac{\partial F}{\partial y'} = t^4, \frac{\partial F}{\partial y''} = t^2, \frac{\partial F}{\partial y'''} = 1, \end{equation*}
        all of which are independent of \(y\) and its derivatives. So the ODE is linear.
      3. \begin{equation*} y' = y + \sin y \end{equation*}
        Solution.
        The highest order of the derivative is 1. So the order is 1.
        Set \(F(t,y, y')= y'-y-\sin y\text{.}\) Then clearly,
        \begin{equation*} \frac{\partial F}{\partial y} = -1+\cos y \end{equation*}
        that depends on \(y\text{.}\) So the ODE is nonlinear.
    3. Check if the given functions \(y(t)\) are solutions to the corresponding ODEs.
      1. \begin{equation*} y(t)= \tan t, y' = 1+ y^2 \end{equation*}
        Solution.
        Since \(y = \tan t\text{,}\) we have \(y' = \sec^2 t\text{.}\)
        Plug in the ODE: \(LHS =\sec^2 t\text{,}\) while \(RHS = 1+\tan^2 t\text{.}\) From the trig identity that \(\sec^2 t = 1+ \tan^2 t\text{,}\) \(LHS=RHS\text{.}\) So \(y = \tan t\) is a solution of the ODE.
      2. \begin{equation*} y(t)= e^t + e^{-2t}, y'' + y' - 2y = 0 \end{equation*}
        Solution.
        \(y = e^t + e^{-2t}\text{,}\)
        \begin{equation*} y' = e^t - 2e^{-2t}, y'' = e^t + 4 e^{-2t}\text{.} \end{equation*}
        \begin{align*} LHS \amp =e^t + 4e^{-2t} + e^t - 2e^{-2t} - 2(e^t + e^{-2t})\\ \amp = e^t (1+1-2) + e^{-2t}(4-2-2) = 0 = RHS, \end{align*}
        \(y = e^t + e^{-2t}\)
      3. \begin{equation*} y(t)=1/t, t^2y'' -2 y = 0 \end{equation*}
        Solution.
        \(y = 1/t = t^{-1}\text{,}\)
        \begin{equation*} y' = -t^{-2}, y'' = -(-2)t^{-3} = 2t^{-3}. \end{equation*}
        \begin{equation*} LHS = t^2 \cdot 2 t^{-3} - 2 t^{-1} = 2t^{-1} - 2t^{-1} = 0\text{.} \end{equation*}
        \(y = 1/t\)
      4. \begin{equation*} y(t)=\sin 2t, y'' + y = \sin 2t \end{equation*}
        Solution.
        \(y = \sin 2t\text{,}\)
        \begin{equation*} y' = \cos 2t \cdot 2 = 2 \cos 2t, y'' = 2 \cdot (-\sin 2t) \cdot 2 = -4 \sin 2t. \end{equation*}
        \begin{equation*} LHS = -4\sin 2t + 2 \cos 2t \neq RHS \text{.} \end{equation*}
        \(y = \sin 2t\)
    4. Find the general solutions to the following ODEs and determine the long term behaviors of the solution.
      1. \begin{equation*} y' + 3y = t + e^{-2t}. \end{equation*}
        Solution.
        \begin{equation*} p(t) = 3, g(t) = t+e^{-2t}. \end{equation*}
        \begin{equation*} \mu(t) = e^{\int p(t) dt} = e^{\int 3 dt} = e^{3t}. \end{equation*}
        Formulate the general solution:
        \begin{equation*} y(t) = \dfrac{\int \mu(t) g(t) dt}{\mu(t)} = \dfrac{\int e^{3t}(t+e^{-2t})dt}{e^{3t}} \end{equation*}
        \begin{align*} \int e^{3t}(t+e^{-2t})dt \amp = \int te^{3t}dt + \int e^{t} dt \\ \amp= t \left(\frac 1 3 e^{3t}\right) - \int 1 \cdot \left(\frac 1 3 e^{3t}\right)dt + e^{t}\\ \amp= \frac {te^{3t}}{3} - \frac 1 9 e^{3t} + e^t + C. \end{align*}
        \begin{align*} y(t) \amp = \dfrac{\frac {te^{3t}}{3} - \frac 1 9 e^{3t} + e^t + C}{e^{3t}}\\ \amp = \dfrac{t}{3} - \frac{1}{9} + e^{-2t} + Ce^{-3t}. \end{align*}
      2. \begin{equation*} (1+t^2) y' + 4t y = (1+t^2)^{-2}. \end{equation*}
        Solution.
        \begin{equation*} y' + \dfrac{4t}{1+t^2} = \frac{1}{(1+t^2)^3}. \end{equation*}
        \begin{equation*} p(t)= \frac{4t}{1+t^2}, g(t) = \frac{1}{(1+t^2)^3}. \end{equation*}
        \begin{equation*} \mu(t) = e^{\int p(t) dt} = e^{\int \frac{4t}{1+t^2}dt} = e^{2\ln (1+t^2)} = (1+t^2)^2 \end{equation*}
        \(\textbf{Warning! Do not abuse algebra! }e^{2\ln u} = u^2, \text{ not }2u!\text{.}\)
        \begin{align*} y(t) \amp= \dfrac{\int \mu(t)g(t)dt}{\mu(t)} = \dfrac{\displaystyle{\int (1+t^2)^2 \cdot \dfrac{1}{(1+t^2)^3} dt}}{(1+t^2)^2}\\ \amp = \dfrac{\displaystyle{\int (1+t^2)^{-1} dt}}{(1+t^2)^2} = \dfrac{\arctan t + C}{1+t^2}. \end{align*}
    5. (Bonus) For the ODE
      \begin{equation*} ty'+ (t-1)y = -e^{-t} \end{equation*}
      Investigate the following sets of initial values
      \begin{equation*} y(0) = 0 \text{ and } y(0) = 1 \end{equation*}
      What happens in each case?
      Solution.
      \begin{equation*} y' + \dfrac{t-1}{t} = -\frac{e^{-t}}{t}. \end{equation*}
      \begin{equation*} p(t) =\dfrac{t-1}{t} = 1 - t^{-1}, g(t) = -\frac{e^{-t}}{t} \end{equation*}
      \begin{equation*} \mu(t)= e^{\int p(t)dt} = e^{\int (1 - t^{-1})dt} = e^{t - \ln t} = e^t \cdot e^{-\ln t} = e^t / t. \end{equation*}
      \(\textbf{Warning! Do not abuse algebra! }e^{-\ln u} = u^{-1} = 1/u, \text{ not }-u!\text{.}\)
      \begin{align*} y(t) \amp= \dfrac{\int e^t/t \cdot (-e^{-t}/ t)dt}{e^t/t} = \frac t {e^t} \cdot \int (-t^{-2})dt \\ \amp = \frac t e^{t}\cdot \left(\frac 1 t+C\right) = e^{-t} + C te^{-t}. \end{align*}
      1. If \(y(0)=0\text{,}\) then \(e^0 + C \cdot 0 e^0 = 0 \Rightarrow 1 = 0\text{.}\) We obtained an equation that can never be satisfied. So the IVP with the initial condition \(y(0)=0\) has no solution.
      2. If \(y(0)=1\text{,}\) then \(e^0 + C \cdot 0 e^0 = 0 \Rightarrow 1 = 1\text{.}\) We obtained an equation that is always satisfied no matter how \(C\) is chosen. So the IVP with the initial condition \(y(0)=1\) has infinitely many solutions.
    6. (no need to write) Review the techniques of integration and make sure you are comfortable to \(u\)-substitution, integrating by parts and integration of \(\cos^2 x, \sin^2 x\text{.}\)
