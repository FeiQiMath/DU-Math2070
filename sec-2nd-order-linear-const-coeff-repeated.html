    Section 2.5 Second-order Linear ODEs with Constant Coefficients - Repeated Real Roots

    Subsection 2.5.1 The formula of the general solution

    Consider the ODE
    \begin{equation*} ay'' + by' + cy = 0 \end{equation*}
    where \(a, b, c\) are constant numbers. We tried the function \(y = e^{rt}\) and got the auxiliary equation
    \begin{equation*} ar^2 + br + c = 0. \end{equation*}
    If the quadratic equation has two repeated roots \(r_1 = r_2 = r,\) in this case, we only obtain one solution
    \begin{equation*} y_1 = e^{rt}. \end{equation*}
    Another solution may be obtained (via cheating) by
    \begin{equation*} y_2 = te^{rt}. \end{equation*}
    The general solution is
    \begin{equation*} y = C_1 e^{rt} + C_2 te^{rt}. \end{equation*}

    Example 2.5.1. \(y'' - 2y' + y = 0.\).

    • Auxiliary equation: \(r^2 - 2r + 1 = 0 \implies r_1 = r_2 = 1.\)
    • General solution: \(y = C_1 e^t + C_2 te^t.\)

    Example 2.5.2. \(4y'' - 4y' + y = 0.\).

    • Auxiliary equation: \(4r^2 - 4r + 1 = 0 \implies r_1 = r_2 = \frac{1}{2}.\)
    • General solution: \(y = C_1 e^{\frac{t}{2}} + C_2 te^{\frac{t}{2}}.\)

    Example 2.5.3. For the IVP \(y'' + 10y' + 25y = 0, \, y(0) = 1, \, y'(0) = \alpha,\)find the critical value of \(\alpha\) where the long-term behavior changes..

    • Auxiliary equation: \(r^2 + 10r + 25 = 0 \implies r_1 = r_2 = -5.\)
    • General solution: \(y = C_1 e^{-5t} + C_2 t e^{-5t}.\)
    • Plug in the initial values:
      \begin{gather*} y(0) = 1 \implies C_1 = 1.\\ y'(0) = \alpha \implies -5C_1 + C_2 = \alpha \implies C_2 = \alpha + 5. \end{gather*}
      Solution of the IVP:
      \begin{equation*} y = e^{-5t} + (\alpha + 5) t e^{-5t}. \end{equation*}
    • As \(t \to \infty,\) the solution \(y \to 0\) no matter how \(\alpha\) is chosen. However, there are two ways approaching zero:
      If \(\alpha \gt -5,\) then \(y \to 0^+.\) If \(\alpha \lt -5,\) then \(y \to 0^-.\)
    Conclusuion: The long-term behavior changes when \(\alpha = -5.\)
    For \(\alpha \gt -5:\) \(y(t) = (1 + (\alpha + 5)t)e^{-5t}, \, \alpha \gt 0.\)
    For \(\alpha \lt -5:\) \(y(t) = (1 + (\alpha + 5)t)e^{-5t}, \, \alpha \lt 0.\)

    Subsection 2.5.2 The method of reduction of order

    How do we know we should multiply by \(t?\) It is a result of a very standard technique called variation of parameters.
    Philosophy: If you know one solution \(y_1(t),\) then since the ODE is linear, for any number \(c,\) the function \(cy_1\) would also be a solution.
    Variation of parameter simply means that replacing the numerical parameter \(c\) with a function \(u(t),\) i.e., set
    \begin{equation*} y_2(t) = u(t)y_1(t). \end{equation*}
    Plugging it back into the ODE, we may get an (usually much simpler) ODE for \(u.\) By solving for \(u,\) we may obtain \(y_2.\)
    In the context of linear homogeneous ODE, the technique is usually referred to as reduction of order.
    Let \(y_1\) be a solution of the homogeneous linear ODE
    \begin{equation*} y'' + p(t)y' + q(t)y = 0. \end{equation*}
    We set \(y_2(t) = u(t)y_1(t)\) and require that \(y_2\) is another solution. This means that
    \begin{equation*} (uy_1)'' + p(uy_1)' + quy_1 = 0 \end{equation*}
    Expanding the left-hand-side:
    \begin{align*} LHS \amp= u''y_1 + 2u'y_1' + uy_1''\\ \amp= u''y_1 + (2y_1' + py_1)u' + pu'y_1 + quy_1\\ \amp= u''y_1 + (2y_1' + py_1)u' + u(y_1'' + py_1' + qy_1)\\ \amp= u''y_1 + (2y_1' + py_1)u' = 0 = RHS \end{align*}
    we obtain a second-order ODE of \(u\) that involves only \(u''\) and \(u'\text{.}\)
    This ODE is essentially a first-order separable ODE of \(u'\text{,}\) i.e.,
    \begin{equation*} y_1 \frac{du'}{dt} = -(2y_1' + py_1)u' \implies \frac{du'}{u'} = -\frac{2y_1' + py_1}{y_1} dt \end{equation*}
    Solving \(u'\) then integrate, we obtain \(u\) and thus \(y_2.\)

    Remark 2.5.4.

    1. When formulating the equation for \(u'\text{,}\) make sure you read the function \(p\) from the standard form.
    2. Since we only need one \(y_2\) that is linearly independent to \(y_1\text{,}\) when solving for \(u'\) and integrating for \(u\text{,}\) we may make a wise choice of the arbitrary constants. It suffices that \(u' \neq 0.\)
    3. It is possible to have a formula for \(u\text{,}\) namely,
      \begin{equation*} u(t) = \int \dfrac{\exp(-\int p(t)dt)}{y_1(t)^2} dt \end{equation*}
      I do not recommend using this formula or memorizing it.

    Example 2.5.5. How to find the \(t\)-factor in the repeated root case.

    Let \(ay'' + by' + cy = 0\) be an ODE whose auxiliary equation has repeated roots \(r_1 = r_2 = r\text{.}\) This means that the ODE can be rewritten as
    \begin{equation*} y'' - 2ry' + r^2y = 0. \end{equation*}
    For \(y_1 = e^{rt}\text{,}\) we set \(y_2 = uy_1\text{,}\) then
    \begin{align*} \amp y_2'' - 2ry_2' + r^2y_2 = (y_1)'' - 2r(y_1)' + r^2uy_1\\ \amp= u''y_1 + 2u'y_1' + uy_1'' - 2r(u'y_1 + uy_1') + r^2uy_1\\ \amp= u''e^{rt} + 2u' \cdot re^{rt} + u \cdot r^2e^{rt} - 2ru'e^{rt} - 2r \cdot re^{rt} + r^2ue^{rt}\\ \amp= u''e^{rt} - 2ru'e^{rt} + r^2ue^{rt} = 0\\ \amp= e^{rt}u'' = 0 \implies u'' = 0 \implies u' = C_1 \end{align*}
    For convenience, we pick \(C_1 = 1\) so that \(u' \neq 0\text{,}\) then integrate to get \(u = t,\) thus
    \begin{equation*} y_2 = te^{rt}. \end{equation*}
    The condition that \(u'\neq 0\) automatically guarantees that \(W(y_1, y_2) \neq 0\) [Why?]. We conclude that
    \begin{equation*} y = C_1e^{rt} + C_2te^{rt} \end{equation*}
    is the general solution.

    Example 2.5.6. For the ODE \(ty'' - y' - 4t^3y = 0,\) knowing that \(y_1 = \sin(t^2)\) is a solution, find the general solution..

    • Standard form:
      \begin{equation*} y'' - \dfrac{1}{t}y' - 4t^2y = 0 \implies p(t) = -\dfrac{1}{t}. \end{equation*}
    • Set \(y_2 = uy_1.\) To formulate the ODE for \(u',\) we have
      \begin{equation*} y_1 = \sin(t^2), \quad y_1' = 2t \cos(t^2) \end{equation*}
      Plug them into the ODE
      \begin{equation*} y_1u'' + (2y_1' + py_1)u' = 0, \end{equation*}
      we have
      \begin{align*} \amp \sin(t^2) \dfrac{du'}{dt} + \left(4t \cos(t^2) - \dfrac{1}{t} \sin(t^2)\right) u' = 0\\ \implies \amp \dfrac{du'}{u'} = \left(-4t \dfrac{\cos(t^2)}{\sin(t^2)} + \dfrac{1}{t}\right) dt \implies \ln|u'| = -2 \ln|\sin(t^2)| + \ln t.\\ \implies \amp u' = \dfrac{t}{\sin^2(t^2)} \implies u = -\dfrac{1}{2} \cot(t^2) \implies y_2 = -\dfrac{1}{2} \cos(t^2). \end{align*}
      By incorporating the \(-1/2\) coefficient into the arbitrary constants, we see that the general solution of the ODE is
      \begin{equation*} y = C_1 \sin(t^2) + C_2 \cos(t^2). \end{equation*}
