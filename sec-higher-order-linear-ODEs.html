    Subsection 2.9.1 General Theory

    For higher-order linear ODEs, many results are similar to those for second-order linear ODEs. We start with the existence and uniqueness theorem.
    ðŸ”—

    Theorem 2.9.1. Existence and Uniqueness Theorem.

    For the \(n\)-th order linear ODE
    \begin{equation*} y^{(n)} + p_1(t)y^{(n-1)} + \cdots + p_{n-1}(t)y' + p_n(t)y = g(t) \end{equation*}
    in its standard form, if \(p_1(t), \ldots, p_n(t)\) and \(g(t)\) are continuous on an open interval \((a,b),\) then for every \(t_0 \in (a,b)\) and real numbers \(y_0, \ldots, y_{n-1},\) there exists a unique solution \(y\) of the ODE on the interval \((a,b)\) that satisfies the initial conditions
    ðŸ”—
    \begin{equation*} y(t_0) = y_0, \quad y'(t_0) = y_1, \quad \cdots, \quad y^{(n-1)}(t_0) = y_{n-1}. \end{equation*}
    ðŸ”—
    We now consider the homogeneous case, i.e., \(g(t) = 0.\)
    ðŸ”—

    Definition 2.9.2.

    For the homogeneous linear ODE
    \begin{equation*} y^{(n)} + p_1(t)y^{(n-1)} + \cdots + p_{n-1}(t)y' + p_n(t)y = 0 \end{equation*}
    on an interval where the existence and uniqueness theorem holds, we say that \(y_1, \ldots, y_n\) form a fundamental set of solutions if they are
    1. solutions of the ODE on the same interval;
      ðŸ”—
    2. linear independent, i.e., the Wronskian
      \begin{equation*} W(y_1, \ldots, y_n) = \begin{vmatrix} y_1 \amp \cdots \amp y_n \\ \vdots \amp \ddots \amp \vdots \\ y_1^{(n-1)} \amp \cdots \amp y_n^{(n-1)} \end{vmatrix} \end{equation*}
      is nowhere vanishing (wonâ€™t require).
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Theorem 2.9.3. Principle of Superposition for Homogeneous ODEs.

    If \(y_1, \ldots, y_n\) form a fundamental set of solutions, then the general solution of the ODE is
    \begin{equation*} y = C_1 y_1 + \cdots + C_n y_n. \end{equation*}
    ðŸ”—
    ðŸ”—
    We now consider the nonhomogeneous case.
    ðŸ”—

    Theorem 2.9.4. Structure of the general solution and the principle of superposition.

    Consider the nonhomogeneous linear ODE
    \begin{equation*} y^{(n)} + p_1(t)y^{(n-1)} + \cdots + p_{n-1}(t)y' + p_n(t)y = g(t) \end{equation*}
    on an open interval where the existence and uniqueness theorem holds.
    ðŸ”—
    1. \begin{equation*} y = y_c + Y, \end{equation*}
      • \(y_c,\) the complementary solution, is the general solution of the homogeneous ODE.
        ðŸ”—
      • \(Y,\) a particular solution, is a solution of the nonhomogeneous ODE.
        ðŸ”—
      ðŸ”—
    2. If \(Y_1\) is a solution of \(y^{(n)} + \cdots + p_n(t)y = g_1(t),\) \(Y_2\) is a solution of \(y^{(n)} + \cdots + p_n(t)y = g_2(t),\) then \(Y_1 + Y_2\) is a solution of \(y^{(n)} + \cdots + p_n(t)y = g_1(t) + g_2(t).\)
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Subsection 2.9.2 Higher order linear homogeneous ODE with constant coefficients

    We first consider the homogeneous case. For the ODE,
    \begin{equation*} a_ny^{(n)} + a_{n-1}y^{(n-1)} + \cdots + a_1y' + a_0y = 0 \end{equation*}
    We try \(y = e^{rt}\) and see that
    \begin{equation*} a_n r^n e^{rt} + a_{n-1} r^{n-1} e^{rt} + \cdots + a_1 r e^{rt} + a_0 e^{rt} = 0. \end{equation*}
    Dividing by \(e^{rt},\) we obtain the auxiliary equation
    \begin{equation*} a_n r^n + a_{n-1} r^{n-1} + \cdots + a_1 r + a_0 = 0. \end{equation*}
    ðŸ”—

    Theorem 2.9.5. Fundamental theorem of algebra (real form).

    Every polynomial with real coefficients can be factored as a product of linear and quadratic polynomials.
    ðŸ”—
    ðŸ”—

    Corollary 2.9.6.

    If \(\alpha + i\beta\) is a root that is repeated \(m\) times, then its conjugate \(\alpha - i\beta\) is also a root that is repeated \(m\) times.
    ðŸ”—
    ðŸ”—
    Let \(r\) be a solution of the auxiliary equation. Then its contribution to the fundamental set of solutions is as follows.
    ðŸ”—
    • Case 1: If \(r\) is a real single root, then it contributes one function \(e^{rt}\) to the fundamental set of solutions.
      ðŸ”—
    • Case 2: If \(r\) is a real root repeated \(m\) times, then it contributes \(m\) functions
      \begin{equation*} e^{rt}, \, te^{rt}, \ldots, \, t^{m-1}e^{rt}, \end{equation*}
      to the fundamental set of solutions.
      ðŸ”—
    • Case 3: If \(r = \alpha + i\beta\) is a complex single root, then together with its conjugate \(\bar{r} = \alpha - i\beta,\) they contribute two functions
      \begin{equation*} e^{\alpha t} \cos \beta t, \, e^{\alpha t} \sin \beta t \end{equation*}
      to the fundamental set of solutions.
      ðŸ”—
    • Case 4: If \(r = \alpha + i\beta\) is a complex root repeated \(m\) times, then together with \(\bar{r} = \alpha - i\beta,\) they contribute 2m functions
      \begin{equation*} e^{\alpha t} \cos \beta t, \, e^{\alpha t} \sin \beta t, \, te^{\alpha t} \cos \beta t, \, te^{\alpha t} \sin \beta t, \, \ldots, \, t^{m-1}e^{\alpha t} \cos \beta t, \, t^{m-1}e^{\alpha t} \sin \beta t, \end{equation*}
      to the fundamental set of solutions.
      ðŸ”—

    Example 2.9.7. \(y''' - y' = 0.\).

    • Auxiliary equation: \(r^3 - r = 0\) where
      \begin{align*} \amp LHS = r(r^2 - 1) = r(r - 1)(r + 1) = 0 \\ \implies \amp r_1 = -1, r_2 = 0, r_3 = 1. \end{align*}
      ðŸ”—
    • General solution:
      \begin{equation*} y = C_1 e^{-t} + C_2 + C_3 e^t. \end{equation*}
      ðŸ”—
    ðŸ”—

    Example 2.9.8. \(y''' - y'' - y' + y = 0\).

    • Auxiliary equation: \(r^3 - r^2 - r + 1 = 0\) where
      \begin{align*} LHS = \ \amp r^2(r - 1) - (r - 1) = (r - 1)(r^2 - 1) \\ = \ \amp (r - 1)(r - 1)(r + 1) = 0 \implies r_1 = -1, r_2 = r_3 = 1. \end{align*}
      ðŸ”—
    • General solution:
      \begin{equation*} y = C_1 e^{-t} + C_2 e^t + C_3 t e^t. \end{equation*}
      ðŸ”—
    ðŸ”—

    Example 2.9.9. \(y''' - 3y'' + 3y - y = 0\).

    • Auxiliary equation: \(r^3 - 3r^2 + 3r - 1 = 0\) where
      \begin{align*} LHS = \ \amp r^3 - 1 - 3r(r - 1) = (r - 1)(r^2 + r + 1) - 3r(r - 1) \\ = \ \amp (r - 1)(r^2 + r + 1 - 3) = (r - 1)(r^2 - 2r + 1) \\ = \ \amp (r - 1)^3 = 0 \implies r_1 = r_2 = r_3 = 1. \end{align*}
      ðŸ”—
    • General solution:
      \begin{equation*} y = C_1 e^t + C_2 te^t + C_3 t^2 e^t. \end{equation*}
      ðŸ”—
    ðŸ”—

    Example 2.9.10. \(y''' - y = 0\).

    • Auxiliary equation: \(r^3 - 1 = 0\) where
      \begin{align*} \amp LHS = (r - 1)(r^2 + r + 1) = 0 \\ \implies \amp r_1 = 1, r_2 = -\dfrac{1}{2} + \dfrac{\sqrt{3}}{2} i, r_3 = -\dfrac{1}{2} - \dfrac{\sqrt{3}}{2} i. \end{align*}
      ðŸ”—
    • General solution:
      \begin{equation*} y = C_1 e^t + C_2 e^{-\frac{t}{2}} \cos\left(\dfrac{\sqrt{3}}{2} t\right) + C_3 e^{-\frac{t}{2}} \sin\left(\dfrac{\sqrt{3}}{2} t\right). \end{equation*}
      ðŸ”—
    ðŸ”—

    Example 2.9.11. \(y^{(4)} + 8y'' + 16y = 0\).

    • Auxiliary equation: \(r^4 + 8r^2 + 16 = 0\) where
      \begin{align*} LHS = \ \amp (r^2 + 4)^2 = 0 \implies r_1 = r_2 = 2i, r_3 = r_4 = -2i. \end{align*}
      ðŸ”—
    • General solution:
      \begin{equation*} y = C_1 \cos 2t + C_2 \sin 2t + C_3 t \cos 2t + C_4 t \sin 2t. \end{equation*}
      ðŸ”—
    ðŸ”—

    Example 2.9.12. \(y^{(4)} + 4y = 0\).

    • Auxiliary equation: \(r^4 + 4 = 0.\) So \(r = (-4)^{1/4}.\) Recall that
      \begin{align*} (-4)^{1/4} = \ \amp (4e^{i(\pi + 2k\pi)})^{1/4} = 4^{1/4} e^{i(\frac{\pi}{4} + \frac{k\pi}{2})}. \end{align*}
      Thus
      \begin{align*} r_1 = \ \amp \sqrt{2} e^{i \frac{\pi}{4}} = \sqrt{2} \left( \cos \frac{\pi}{4} + i \sin \frac{\pi}{4} \right) = 1 + i,\\ r_2 = \ \amp \sqrt{2} e^{i \frac{3\pi}{4}} = \sqrt{2} \left( \cos \frac{3\pi}{4} + i \sin \frac{3\pi}{4} \right) = -1 + i,\\ r_3 = \ \amp \sqrt{2} e^{i \frac{5\pi}{4}} = \sqrt{2} \left( \cos \frac{5\pi}{4} + i \sin \frac{5\pi}{4} \right) = -1 - i,\\ r_4 = \ \amp \sqrt{2} e^{i \frac{7\pi}{4}} = \sqrt{2} \left( \cos \frac{7\pi}{4} + i \sin \frac{7\pi}{4} \right) = 1 - i. \end{align*}
      ðŸ”—
    • General solution:
      \begin{equation*} y = C_1 e^t \cos t + C_2 e^t \sin t + C_3 e^{-t} \cos t + C_4 e^{-t} \sin t. \end{equation*}
      ðŸ”—
    ðŸ”—

    Subsection 2.9.3 Higher order linear nonhomogeneous ODE with constant coefficients

    We now consider the nonhomogeneous case. For the ODE
    \begin{equation*} a_ny^{(n)} + a_{n-1}y^{(n-1)} + \cdots + a_1y' + a_0y = g(t), \end{equation*}
    Variation of parameters would be quite complicated. But the method of coefficients works in the same way.
    ðŸ”—
    • Step 1: Separate \(g(t)\) according to exponential coefficients, i.e., write \(g(t) = g_1(t) + g_2(t) + \cdots + g_m(t),\) where each \(g_j(t)\) is of the form:
      \begin{equation*} g_j(t) = e^{\alpha_j t} \cos \beta_j t \, (\text{polynomial}) + e^{\alpha_j t} \sin \beta_j t \, (\text{polynomial}), \end{equation*}
      For example, if \(g(t) = te^t + 3 \cos 3t + t^2 + 4t \sin 3t + 2t^2 e^t + 1\) we should write \(g(t) = g_1(t) + g_2(t) + g_3(t),\) where
      \begin{equation*} g_1(t) = (2t^2 + t)e^t, \, g_2(t) = 3 \cos 3t + 4t \sin 3t, \, g_3(t) = t^2 + 1. \end{equation*}
      ðŸ”—
    • Step 2: We find a particular solution \(Y_j\) for each
      \begin{equation*} g_j(t) = e^{\alpha_j t} \cos \beta_j t \, (\text{polynomial}) + e^{\alpha_j t} \sin \beta_j t \, (\text{polynomial}). \end{equation*}
      ðŸ”—
      • The first ansatz should be set as
        \begin{equation*} Y_j = e^{\alpha_j t} \cos \beta_j t (A_n t^n + \cdots + A_0) + e^{\alpha_j t} \sin \beta_j t (B_n t^n + \cdots + B_0). \end{equation*}
        where \(n\) is the largest degree appearing in the polynomial.
        ðŸ”—
      • If the exponential coefficient \(\alpha_j + i\beta_j\) does not appear as a root of the auxiliary equation, the first ansatz succeeds. If \(\alpha + i\beta\) is a root repeated \(m\) times, the final ansatz should be modified by \(t^m,\) i.e.,
        \begin{equation*} Y = t^m \, (\text{First Ansatz}). \end{equation*}
        ðŸ”—
      ðŸ”—
    • Step 3: Once we find a particular solution for each \(g_j(t)\text{,}\) the principle of superposition implies that
      \begin{equation*} Y = Y_1 + Y_2 + \cdots + Y_m \end{equation*}
      is a particular solution of the nonhomogeneous ODE.
      \begin{equation*} a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y = g(t) \end{equation*}
      for \(g(t) = g_1(t) + g_2(t) + \cdots + g_m(t).\)
      ðŸ”—

    Remark 2.9.13.

    If you wish to ask the computer to carry out the calculation, make sure you plug in the final ansatz for each \(Y_j\) in \(Y.\)

    Example 2.9.20. Find the general solution of \(y''' - y'' - y' + y = 8e^t + 4e^{-t} + 25 \cos 2t + t^2\).

    1. Auxiliary equation is
      \begin{align*} \amp r^3 - r^2 - r + 1 = r^2(r - 1) - (r - 1) \\ = \ \amp (r - 1)(r^2 - 1) = (r - 1)^2(r + 1) = 0. \end{align*}
      Solutions are \(r_1 = -1, r_2 = 1, r_3 = 1.\)
      ðŸ”—
      Complementary solution:
      \begin{equation*} y_c = C_1 e^{-t} + C_2 e^t + C_3 t e^t. \end{equation*}
      ðŸ”—
      ðŸ”—
    2. Let \(Y_1\) be a solution of \(y''' - y'' - y' + y = 8e^t.\)
      ðŸ”—
      • First ansatz \(Y_1 = Ae^t.\) Exp. Coeff. \(= 1\) which is a double root. Final ansatz
        \begin{equation*} Y_1 = At^2 e^t. \end{equation*}
        ðŸ”—

        \begin{align*} Y'_1 = \ \amp 2Ate^t + At^2 e^t,\\ Y''_1 = \ \amp 2Ae^t + 2Ate^t + 2Ate^t = 2Ae^t + 4Ate^t + At^2 e^t,\\ Y'''_1 = \ \amp 2Ae^t + 4A(e^t + te^t) + 2Ate^t = 6Ae^t + 6Ate^t. \end{align*}
        ðŸ”—
      • Plug the derivatives into the LHS,
        \begin{align*} \amp Y'''_1 - Y''_1 - Y'_1 + Y_1 \\ = \ \amp 6Ae^t + 6Ate^t + At^2 e^t - (2Ae^t + 4Ate^t + At^2 e^t)\\ \amp - (2Ate^t + At^2 e^t) + At^2 e^t = 4Ae^t = 8e^t \implies A = 2. \end{align*}
        ðŸ”—
      So
      \begin{equation*} Y_1 = 2t^2 e^t. \end{equation*}
      ðŸ”—
      ðŸ”—
    3. Let \(Y_2\) be a solution of \(y''' - y'' - y' + y = 4e^{-t}.\)
      ðŸ”—
      • First ansatz \(Y_2 = Be^{-t}.\) Exp. Coeff. \(= -1\) which is a single root. Final ansatz
        \begin{equation*} Y_2 = Bte^{-t}. \end{equation*}
        ðŸ”—
      • We calculate the derivatives:
        \begin{align*} Y'_2 = \ \amp Be^{-t} - Bte^{-t},\\ Y''_2 = \ \amp -Be^{-t} - (Be^{-t} - Bte^{-t}) = -2Be^{-t} + Bte^{-t},\\ Y'''_2 = \ \amp 2Be^{-t} + Be^{-t} - Bte^{-t} = 3Be^{-t} - Bte^{-t}. \end{align*}
        ðŸ”—
      • Plug the derivatives into the LHS,
        \begin{align*} \amp Y'''_2 - Y''_2 - Y'_2 + Y_2 \\ = \ \amp 3Be^{-t} - (-2Be^{-t} + Bte^{-t}) - (-Be^{-t} - Bte^{-t})\\ = \ \amp - (Be^{-t} - Bte^{-t}) + Bte^{-t} = 4Be^{-t} = 4e^{-t} \implies B = 1. \end{align*}
        ðŸ”—
      So
      \begin{equation*} Y_2 = te^{-t}. \end{equation*}
      ðŸ”—
      ðŸ”—
    4. Let \(Y_3\) be a solution of \(y''' - y'' - y' + y = 25 \cos 2t.\)
      ðŸ”—
      • First ansatz \(Y_3 = C \cos 2t + D \sin 2t.\) Exp. Coeff. \(= 2i\text{,}\) not a root. Final ansatz is the first ansatz.
        ðŸ”—
      • We calculate the derivatives:
        \begin{align*} Y'_3 = \ \amp -2C \sin 2t + 2D \cos 2t,\\ Y''_3 = \ \amp -4C \cos 2t - 4D \sin 2t,\\ Y'''_3 = \ \amp 8C \sin 2t - 8D \cos 2t. \end{align*}
        ðŸ”—
      • Plug the derivatives into the LHS,
        \begin{align*} \amp Y'''_3 - Y''_3 - Y'_3 + Y_3 \\ = \ \amp 8C \sin 2t - 8D \cos 2t - (-4C \cos 2t - 4D \sin 2t)\\ \amp - (-2C \sin 2t + 2D \cos 2t) + C \cos 2t + D \sin 2t\\ = \ \amp \sin 2t (10C + 5D) + \cos 2t (-10D + 5C) = 25 \cos 2t\\ \implies \amp 10C + 5D = 0, -10D + 5C = 25 \implies C = 1, D = -2. \end{align*}
        ðŸ”—
      So
      \begin{equation*} Y_3 = \cos 2t - 2 \sin 2t. \end{equation*}
      ðŸ”—
