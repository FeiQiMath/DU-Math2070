    Section 2.2 Second-order Linear ODEs with Constant Coefficients - Real Roots

    Subsection 2.2.1 Auxiliary Equation

    Consider the ODE
    \begin{equation*} ay'' + by' + cy = 0 \end{equation*}
    where \(a, b, c\) are constant numbers. This particular type of ODE can be solved, and are widely used in science and engineering.
    To solve this ODE, we try the function \(y = e^{rt}.\) Then
    \begin{equation*} a(e^{rt})'' + b(e^{rt})' + ce^{rt} = (ar^2 + br + c)e^{rt} = 0. \end{equation*}
    This means that if \(y = e^{rt}\) is a solution, then necessarily, \(r\) is a root of the quadratic equation
    \begin{equation*} ar^2 + br + c = 0. \end{equation*}
    We call it the auxiliary equation.

    Subsection 2.2.2 The case of two distinct real roots

    If the auxiliary equation has two distinct real roots \(r_1, r_2\text{,}\) then
    \begin{equation*} y_1 = e^{r_1 t}, \, y_2 = e^{r_2 t} \end{equation*}
    are solutions, with \(W(y_1, y_2) \neq 0\) (see ExampleÂ 2.1.11). Then by the principle of superpositionÂ 2.1.4,
    \begin{equation*} y = C_1 e^{r_1 t} + C_2 e^{r_2 t} \end{equation*}
    is the general solution.

    Example 2.2.1. \(y'' - 2y' - 3y = 0.\).

    Auxiliary equation: \(r^2 - 2r - 3 = 0 \implies r_1 = 3, \, r_2 = -1.\)
    General solution: \(y = C_1 e^{3t} + C_2 e^{-t}.\)

    Example 2.2.2. \(y'' - 5y' + 6y = 0.\).

    Auxiliary equation: \(r^2 - 5r + 6 = 0 \implies r_1 = 2, \, r_2 = 3.\)
    General solution: \(y = C_1 e^{2t} + C_2 e^{3t}.\)

    Example 2.2.3. \(y'' - 5y' - 6y = 0.\).

    Auxiliary equation: \(r^2 - 5r - 6 = 0 \implies r_1 = 6, \, r_2 = -1.\)
    General solution: \(y = C_1 e^{6t} + C_2 e^{-t}.\)

    Example 2.2.4. \(2y'' - 7y' + 3y = 0.\).

    Auxiliary equation: \(2r^2 - 7r + 3 = 0 \implies r_1 = \frac{1}{2}, \, r_2 = 3.\)
    General solution: \(y = C_1 e^{t/2} + C_2 e^{3t}.\)

    Subsection 2.2.3 Initial Value Problems and Long-term behaviors

    Knowing the general solution, we can solve initial value problems and analyze the long-term behavior of the solution.

    Example 2.2.5. \(y'' - 4y' - 6y = 0, \, y(0) = 1, \, y'(0) = 0.\) Find the solution and determine the long-term behavior..

    • Auxiliary equation: \(r^2 - 4r - 6 = 0 \implies r = 2 \pm \sqrt{10}.\)
    • General solution: \(y = C_1 e^{(2+\sqrt{10})t} + C_2 e^{(2-\sqrt{10})t}.\)
    • From the initial values:
      \begin{equation*} y(0) = 1 \implies C_1 + C_2 = 1 \end{equation*}
      \begin{equation*} y'(0) = 1 \implies (2 + \sqrt{10})C_1 + (2 - \sqrt{10})C_2 = 0. \end{equation*}
      Solving the system, we obtain
      \begin{equation*} C_1 = \frac{-5 - \sqrt{10}}{10}, \, C_2 = \frac{5 + \sqrt{10}}{10}. \end{equation*}
    So the solution is
    \begin{equation*} y = \frac{-5 - \sqrt{10}}{10} e^{(2+\sqrt{10})t} + \frac{5 + \sqrt{10}}{10} e^{(2-\sqrt{10})t}. \end{equation*}
    As \(t \to \infty\text{,}\) the second term approaches zero. The solution is dominated by the first term, which approaches \(- \infty.\)

    Example 2.2.6. \(y'' - 5y' + 6y = 0, \, y(0) = \alpha, \, y'(0) = 1.\) Determine the critical value of \(\alpha\) where the long-term behavior changes..

    • Auxiliary equation: \(r^2 - 5r + 6 = 0 \implies r_1 = 2, \, r_2 = 3.\)
    • General solution: \(y = C_1 e^{2t} + C_2 e^{3t}.\)
    • From the initial values:
      \begin{equation*} y(0) = \alpha \implies C_1 + C_2 = \alpha \end{equation*}
      \begin{equation*} y'(0) = 1 \implies 2C_1 + 3C_2 = 1. \end{equation*}
      Solving the system, we obtain
      \begin{equation*} C_1 = 3\alpha - 1, \, C_2 = 1 - 2\alpha. \end{equation*}
    So the solution is
    \begin{equation*} y = (3\alpha - 1)e^{2t} + (1 - 2\alpha)e^{3t}. \end{equation*}
    As \(t \to \infty\text{,}\) the solution is dominated by the second term. Note that
    \begin{gather*} 1 - 2\alpha \gt 0 \implies \lim_{t \to \infty} y(t) = +\infty,\\ 1 - 2\alpha \lt 0 \implies \lim_{t \to \infty} y(t) = -\infty. \end{gather*}
    Thus, the long-term behavior changes when \(1 - 2\alpha = 0 \implies \alpha = \frac{1}{2}.\)
